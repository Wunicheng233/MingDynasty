# 04 - 时间与移动系统

## 一、时间系统（太阁4核心节奏）

### 1.1 时间单位

| 单位 | 说明 |
|------|------|
| **天** | 基本时间单位，绝大多数行动消耗天数 |
| **月** | 30天 = 1月，**奇数月1日自动召开评定会**（太阁是每月一次，这里改为两月一次契合明朝军政节奏） |
| **年** | 12月 = 1年，记录历史年份和角色年龄 |

### 1.2 时间推进规则

- **主动推进**：执行行动后自动增加对应天数
- **被动跳跃**：剧情事件可直接跳跃数月（如"数月后……"）
- **时间暂停**：菜单、对话、战斗中时间不流逝

### 1.3 明朝纪年显示

为增强沉浸感，同时显示**年号纪年**和**公历**：

```
═══════════════════════════════════════════
  洪武元年（1368年）正月初一 · 春季
  应天府 · 未时
═══════════════════════════════════════════
```

**纪年转换规则**：
- 1356年以前：元朝年号（至正）
- 1356-1367：吴国公元年
- 1368年以后：洪武年号

### 1.4 季节与天气（可选扩展）

- **四季**：春（1-3）夏（4-6）秋（7-9）冬（10-12）
- 天气影响：
  - 雨天：火器威力减半
  - 大雪：移动时间+20%
  - 顺风：水战火攻+20%伤害

### 1.5 时间驱动事件

| 事件类型 | 触发规则 |
|----------|---------|
| **评定会** | 奇数月1日，玩家在居城则自动触发 |
| **新年朝贺** | 每年1月1日，领取俸禄，年龄增长 |
| **科举乡试** | 每年八月 |
| **武举会试** | 每年二月 |
| **历史剧情** | 按预定日期自动触发（如郭子兴病逝1355年） |

**优先级**：主线事件 > 支线事件 > 定期事件 > 随机事件

## 二、移动系统

### 2.1 城镇网络

游戏世界由若干城镇连接而成，每个城镇有唯一 `city_id`，城镇间预设连接和基础移动天数。

**城镇等级**：
| 等级 | 规模 | 代表 |
|------|------|------|
| 都城级 | 3万人 | 应天府、大都 |
| 省会级 | 2万人 | 苏州、杭州 |
| 府城级 | 1.2万人 | 扬州、镇江 |
| 县城级 | 6千人 | 句容、丹徒 |

### 2.2 移动时间计算

```
实际天数 = 基础天数 × (1 - 骑战技能等级 × 10%) × 季节修正
```

- 骑战技能每级减少10%移动时间，最少1天
- 冬季移动时间+20%
- 城镇内切换设施不消耗时间

**移动界面示例**：
```
═══════════════════════════════════════════
            大地图 · 移动
═══════════════════════════════════════════
当前位置：应天府
可前往：
  1. 凤阳府（4天 → 骑战Lv2缩短至3天）
  2. 苏州府（3天 → 骑战Lv2缩短至2天）
  3. 镇江府（2天 → 骑战Lv2缩短至1天）
  4. 返回
═══════════════════════════════════════════
```

### 2.3 移动中的随机事件

移动过程中有10%概率触发随机事件：
- 遭遇山贼 → 触发个人战，胜利得小钱
- 行商叫卖 → 可低价买稀有宝物
- 路遇难民 → 可选择施舍（+魅力）或无视
- 义士投奔 → 魅力高可招募

## 三、时间限制与策略性（借鉴太阁4）

| 限制类型 | 说明 | 后果 |
|----------|------|------|
| 主命期限 | 接受的主命有完成天数上限 | 超期扣功勋，降亲密度 |
| 师父云游 | 某些师父只在特定月份停留 | 错过等半年 |
| 人物寿命 | NPC有生卒年 | 到期可能死亡 |
| 剧情窗口 | 某些事件只在特定时间段触发 | 错过本周目无法触发 |

这种时间限制给玩家带来**"在有限时间内平衡选择"**的策略乐趣——是先修行提升技能，还是先赶去完成主命攒功勋？

## 四、人物动态移动（太阁4AI特色）

不仅仅玩家会移动，NPC也会：

1. **指令性移动**：NPC执行主命前往目的地
2. **规律性移动**：云游师父定期在几个城市间移动
3. **事件驱动移动**：随着剧情推进，人物改变归属地

玩家在路上可能偶遇同样在移动的NPC，触发随机对话或事件。

## 五、数据结构

### 5.1 全局时间

```json
{
  "year": 1356,
  "month": 3,
  "day": 15,
  "total_days": 4925
}
```

`total_days` 是从起始日期累计的总天数，方便计算时间差。

### 5.2 城镇连接数据

```json
{
  "city_id": "YINGTIAN",
  "name": "应天府",
  "connections": [
    { "target": "FENGYANG", "base_days": 4 },
    { "target": "SUZHOU", "base_days": 3 }
  ]
}
```

## 六、与太阁立志传4的对比

| 太阁4 | 洪武立志传 |
|------|------------|
| 每月评定 | 每两月评定（契合明朝军政） |
| 京都/分国/郡/城结构 | 都城/省会/府城/县城四级 |
| 马儿减少移动时间 | 骑战技能减少移动时间 |
| 随机路上事件 | 随机路上事件（本土化） |

核心机制完全继承，只是地理和行政层级适配明朝。
