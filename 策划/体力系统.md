## 《洪武立志传》体力系统详细设计

体力系统是《洪武立志传》中**角色生存与行动的资源管理者**，它直接关联玩家的每一项行动决策，是驱动游戏节奏、制造策略张力的核心机制。借鉴《太阁立志传4》的经典设计——体力过低会导致无法执行部分指令、修行效率下降甚至病倒——并结合明朝初年的社会风貌，我们将其本土化为一个既真实又富有趣味的系统。

以下从体力基础定义、消耗规则、恢复规则、伤病状态层级、与其他系统的耦合、数据与界面实现六个层面详细阐述。


### 一、体力基础定义

#### 1.1 体力值（Vitality，简称 VIT）

体力是角色的生命活力与身体状态的量化指标，范围 **0 ~ 最大体力**。

- **当前体力（current_vit）**：实时变动的数值，归零则角色昏迷/重病，触发负面事件。
- **最大体力（max_vit）**：由基础值 + 加成构成，决定了体力的上限。

#### 1.2 最大体力计算公式

```
最大体力 = 基础体力 + 称号加成 + 宝物加成
基础体力 = 60 + (武力值 × 0.8)
```

**示例**：
- 朱元璋武力 78 → 基础体力 = 60 + 62 = 122
- 常遇春武力 100 → 基础体力 = 60 + 80 = 140

**称号加成**（仅佩戴中的主称号生效）：
| 称号 | 体力上限加成 |
|------|------------|
| 乡勇 | +5 |
| 壮士 | +10 |
| 豪侠 | +15 |
| 宗师 | +20 |
| 武圣 | +30 |
| 天下第一 | +50 |

**宝物加成**：装备特定宝物卡（武器、防具、丹药）可增加最大体力，例如：
- 玄铁甲：+15 最大体力
- 武当内丹：+10 最大体力
- 千年灵芝：永久 +5 最大体力（消耗品）

#### 1.3 体力状态层级

体力不仅是数字，更通过五个状态层级影响角色能力：

| 体力百分比 | 状态名称 | 图标示意 | 效果描述 |
|-----------|---------|---------|---------|
| 100% ~ 75% | **精力充沛** | ❤️❤️❤️❤️❤️ | 无任何负面影响，修行效率 +10% |
| 75% ~ 40% | **略有倦意** | ❤️❤️❤️❤️🤍 | 移动速度 -10%，个人战先手率微降 |
| 40% ~ 20% | **疲惫不堪** | ❤️❤️❤️🤍🤍 | 移动速度 -20%，修行经验获取 -30%，个人战手牌数 -1（最低保留4张） |
| 20% ~ 5% | **体力透支** | ❤️❤️🤍🤍🤍 | 无法进行修行、出征、长距离移动；个人战武力 -20%，容易失误 |
| < 5% | **危在旦夕** | ❤️🤍🤍🤍🤍 | 无法执行绝大多数指令；每过1天有10%概率触发“昏迷”事件，损失大量时间并可能死亡（若无人救助） |

**特殊状态**：
- **生病**：独立于体力百分比的异常状态，由事件或长期透支引发。生病期间体力上限临时 -30%，且自然恢复停止，必须通过医馆治疗或服用药物解除。


### 二、体力消耗规则

#### 2.1 日常行动消耗

| 行动类别 | 基础体力消耗 | 受影响因素 |
|---------|------------|-----------|
| 城镇内设施间移动 | 0 | — |
| 修行（每次5天） | -8 | 技能等级每高1级多消耗2点（Lv3修行消耗12点） |
| 执行政务类主命（全程） | -10 ~ -30 | 任务难度和时长决定，工政/农政技能高可减少消耗 |
| 执行潜入/破坏主命 | -15 ~ -25 | 密探技能高可减少消耗 |
| 长距离移动（跨省） | -5 / 次 | 骑战技能每等级减少1点消耗，最低消耗2点 |
| 宴饮/茶会 | -3 | — |
| 送礼/交谈 | 0 | — |
| 赌博 | -2 / 次 | — |
| 武馆切磋 | 战斗损失体力的50% | 实际扣除量 = (战斗前体力 - 战斗后体力) × 0.5 |

#### 2.2 个人战体力消耗

个人战中体力作为战斗资源实时变化，战斗结束后按规则结算：

- **剧情个人战**：胜负可能强制改变体力（如战败则体力降为1，触发后续剧情）。
- **切磋/遭遇战**：战斗结束后，若胜利，玩家最终体力 = 战斗结束时剩余体力 + (战斗损失体力 × 0.3)；若失败，则体力降至战斗结束时的真实值，并可能进入“体力透支”状态。

**个人战体力恢复技能**（如“调息回元”）：
- 使用后当场恢复一定体力，但该恢复量不计入战后折算。

#### 2.3 合战体力消耗

合战中主将体力并不直接作为战斗资源，但：

- 出征前若主将体力低于40%，无法领兵出征。
- 长期围城（超过15天），主将每5天自动扣除3点体力（代表风餐露宿）。
- 合战中被“狙杀敌将”战术卡击中，主将体力额外 -20。

#### 2.4 时间流逝自然消耗

- 每经过 **10天**（无论进行何种行动），体力自然 -1，代表岁月与奔波带来的身体损耗。此消耗在“精力充沛”状态下减半（每20天-1）。


### 三、体力恢复规则

#### 3.1 主动恢复方式

| 方式 | 恢复量 | 消耗 | 条件/限制 |
|------|-------|------|----------|
| **旅店住宿** | 恢复至满体力 | 金钱：50文 × 恢复百分比 | 消耗5天时间；生病状态无法通过旅店完全恢复 |
| **自宅休养** | 恢复至满体力 | 无 | 消耗5天；仅限拥有自宅时 |
| **医馆治疗** | 恢复至满体力 + 解除生病状态 | 金钱：200文（含药费） | 消耗3天；医术技能高的医师恢复更快 |
| **使用药品** | 金疮药 +30；续命丹 +50；九转还魂丹 +100 | 消耗物品 | 可在任何非战斗状态使用；部分药品有抗药性（连续使用效果递减） |
| **与妻子互动** | +15 | 无 | 每天限1次；仅限已婚且妻子在自宅中 |
| **寺庙布施** | +10（并加福缘） | 金钱：100文 | 每天限1次 |
| **特殊事件** | 不定 | — | 如“路遇游医”、“品尝仙果”等随机事件 |

#### 3.2 被动恢复规则

- **自然恢复**：在非生病状态下，每经过 **1天**，体力自动恢复 **2点**。若在旅店/自宅休养期间，每天恢复量提升至 **5点**。
- **睡眠恢复**：每次住宿/休养结束时，额外获得“睡眠充足”增益，接下来3天内自然恢复速度翻倍。
- **称号影响**：佩戴“武圣”以上称号，自然恢复速度 +50%。

#### 3.3 生病状态下的恢复

- 生病时**自然恢复停止**。
- 必须前往**医馆**接受“诊脉开方”治疗（消耗3天，200文），或使用“祛病丸”（消耗品）方可解除生病状态。
- 生病期间体力上限临时降低30%，解除后恢复。


### 四、伤病与死亡机制

#### 4.1 重伤昏迷

当体力降至 **0** 时：
- 若在城镇内，自动触发“被路人救起”事件，消耗 **7天** 时间，体力恢复至20%，并进入“体力透支”状态。
- 若在野外/行军途中，触发“被部下/同伴救起”事件（需有同行者），同样消耗时间；若孤身一人，则有 **30%概率直接死亡**（游戏结束），70%概率被当地村民救起，但损失当前持有金钱的50%。

#### 4.2 久病不愈

若生病状态持续 **超过30天** 未治疗：
- 最大体力永久 -5（可叠加）。
- 魅力值临时 -10（因病容憔悴）。
- 可能触发“病情加重”事件，转为“重病”状态，需更长时间治疗。

#### 4.3 角色死亡

- 玩家角色死亡后，若有子嗣（需年满15岁）或结义兄弟，可选择由其继承，继续游戏。
- 继承后，继承者的属性、技能、卡片部分继承（具体规则另表），但体力上限会基于继承者自身武力重新计算。
- 若无继承人，则游戏结束，返回标题画面。


### 五、与其他系统的耦合关系

体力系统作为核心资源层，与几乎所有主要系统产生交互：

| 系统 | 耦合方式 | 数据流向 |
|------|---------|---------|
| **属性系统** | 武力决定最大体力基础值 | 武力 → 最大体力 |
| **技能系统** | 骑战减少移动消耗；医术提升药品效果和恢复技威力；工政/农政减少政务消耗 | 技能 → 体力消耗修正 |
| **个人战系统** | 战斗实时消耗体力；低体力影响手牌数和武力；恢复技当场回复 | 体力 ↔ 个人战 |
| **合战系统** | 低体力无法出征；长期围城消耗体力；战术卡可削减敌将体力 | 体力 → 合战限制 |
| **主命系统** | 执行主命消耗体力；低体力无法接取部分主命 | 体力 → 主命权限 |
| **修行系统** | 每次修行固定消耗体力；低体力降低修行经验获取 | 体力 → 修行效率 |
| **移动系统** | 长距离移动消耗体力；低体力限制移动速度 | 体力 ↔ 移动 |
| **社交系统** | 低体力/生病状态NPC可能拒绝互动；与妻子互动可恢复体力 | 体力 ↔ 社交 |
| **时间系统** | 时间流逝自然消耗体力；恢复方式消耗天数 | 时间 ↔ 体力 |
| **设施系统** | 旅店、自宅、医馆、寺庙提供体力恢复服务 | 设施 → 体力恢复 |
| **事件系统** | 随机事件可能消耗或恢复体力；剧情事件可强制改变体力 | 事件 ↔ 体力 |
| **经济系统** | 恢复体力需要消耗金钱（住店、医馆、布施） | 经济 → 体力恢复 |


### 六、明朝特色设计

#### 6.1 过劳与勤政

作为君主，若连续多日不休息（如连续处理政务超过20天），会累积“勤政过劳”隐藏值。当隐藏值满时，触发“勤政猝晕”事件，强制休息7天，并可能降低健康上限。这是对朱元璋“工作狂”历史形象的致敬与警示。

#### 6.2 丹药与长生

明朝多位皇帝迷信丹药。游戏后期可通过道士获得“仙丹”，服用后瞬间恢复全部体力，且获得临时增益，但每次服用会增加“丹毒”值。丹毒累计过高会导致最大体力永久下降，甚至触发“暴毙”事件——契合嘉靖、万历等朝的现实。

#### 6.3 缠足与女性体力

明朝女性缠足普遍，女性角色（如马秀英、女儿）的初始体力上限较男性低10%，且移动消耗增加。但可通过事件“放足”改善，这是对历史背景的写实处理。

#### 6.4 军屯与体力

若玩家为武将，在驻地“军屯”期间，士兵训练度与自身体力挂钩。体力充沛时训练效果更佳，形成“将强则兵强”的正向循环，呼应明朝卫所制的屯田养兵策略。


### 七、数据与界面实现

#### 7.1 数据结构（JSON）

```json
{
  "player_vitality": {
    "current": 98,
    "max": 122,
    "status": "精力充沛",
    "is_sick": false,
    "overwork_counter": 0,
    "drug_toxicity": 0
  }
}
```

#### 7.2 界面显示

主状态栏常驻显示：
```
══════════════════════════════════════════════════════════════
  [洪武元年三月十五]  应天府  银两：320贯  体力：❤️❤️❤️❤️🤍 (98/122)
══════════════════════════════════════════════════════════════
```

点击体力图标可展开详细状态界面：
```
═══════════════════════════════════════════
            身体状况 · 自省
═══════════════════════════════════════════
当前体力：98 / 122（精力充沛）
生病状态：无    丹毒累积：0

【体力详情】
基础体力：60 + 62（武力78）= 122
称号加成：豪侠 +15
宝物加成：玄铁甲 +15
───────────────────────────
合计上限：152（当前未佩戴豪侠，故为122）

【恢复预测】
· 旅店住宿5天：恢复至满（花费约60文）
· 自然恢复：每天+2点

1. 使用药品  2. 查看状态影响  3. 返回
═══════════════════════════════════════════
```

#### 7.3 行动前体力提示

当玩家选择消耗体力较大的行动时，系统会给出预警：
```
你当前体力为 32/122（疲惫不堪），执行此主命预计消耗15体力，
完成后可能进入“体力透支”状态。是否继续？(Y/N)
```

#### 7.4 体力自动检查

每推进一天，系统自动检查体力相关事件：
- 若体力归零 → 触发昏迷事件
- 若生病且未治疗超过30天 → 触发加重事件
- 若过劳隐藏值满 → 触发勤政猝晕事件
