## 《洪武立志传》个人战完整机制·本土化方案

以下完全基于《太阁立志传4》核心框架，全面本土化为明朝初年背景。个人战不仅是武艺切磋，更是“武举应试”、“江湖闯荡”、“阵前单挑”的核心玩法。


### ⚔️ 一、个人战基础机制

**触发场景**：
- 道场/武馆“习武”（修行小游戏）
- 向武将请求“切磋”
- 遭遇山贼、刺客
- 武举乡试/会试/殿试的武艺考核
- 剧情单挑（如朱元璋单挑陈友谅部将）

**参战属性**：
- **体力**：初始值由武力决定（武力×2 + 50），归零则战败
- **手牌数**：由“武艺”技能等级决定
  - 武艺Lv0：4张
  - 武艺Lv1：5张
  - 武艺Lv2：6张
  - 武艺Lv3：7张

**回合流程**：
1. **抽牌阶段**：从个人牌堆抽至手牌上限
2. **出牌阶段**：选择出牌
   - **特殊技卡**（红色）：每回合最多出 **1张**
   - **数字卡**（1~9）：每回合最多出 **3张**
   - 可只出特殊技卡，也可只出数字卡
3. **判定阶段**：比较双方招式“优先度”，高者先攻击
   - 若招式类型相同，比较组合强度（三同张 > 三连张 > 无组合）
   - 若完全同型，比较数字之和
4. **结算阶段**：计算伤害，扣除体力，进入下一回合

**胜负条件**：
- 一方体力归零则败北
- 主动认输（消耗一回合）
- 特殊剧情可强制中断


### 🃏 二、牌组构成

每位角色拥有独立的 **“个人战牌堆”**，由三部分构成：

| 类型 | 说明 | 获取方式 |
|------|------|---------|
| **数字卡** | 1~9，共9张基础卡，永远在手牌中 | 初始拥有 |
| **特殊技卡（红卡）** | 主动使用的武技，每场战斗可反复抽取 | 武馆习得、击败对手、秘笈 |
| **秘传卡（金卡）** | 传奇武将专属，威力极大 | 获得对应人物卡后，在特定条件习得 |

**手牌上限与武艺等级**：武艺Lv越高，手牌越多，组合可能性越大，这是修行的核心收益。

### 💡 十、开发实现要点

1. **优先度判定逻辑**（伪代码）：
   ```
   function comparePriority(myMove, enemyMove):
       if myMove.type == "secret" and enemyMove.type != "secret": return mine
       if myMove.type == "special" and enemyMove.type == "combo": return mine
       if both are combo:
           if myMove.pattern == "triple" and enemyMove.pattern == "straight": return mine
           if same pattern: return higher_sum
   ```

2. **数字组合识别**：对玩家出的数字卡排序后，匹配预设组合列表。建议先检查是否四张组合（如1-3-6-8），再检查三张组合。

3. **牌堆洗牌机制**：个人牌堆由数字卡（固定9张）+ 特殊技卡（玩家已习得的所有红卡）+ 秘传卡（已习得的所有金卡）组成。每场战斗开始时洗牌，每回合抽至手牌上限。

4. **体力与武力公式**：
   - 最大体力 = 武力×2 + 50
   - 基础伤害 = 武力/5 + 招式威力系数×10（浮动）

5. **文字游戏表现**：重点写好每个招式的“出招描述”和“击中反馈”，这是沉浸感的核心。建议为每张卡配2~3套随机文本，避免重复。


这套个人战系统完全继承了《太阁立志传4》的策略深度，同时通过数字组合技、秘传卡和明朝背景的深度融合，形成了《洪武立志传》独有的战斗体验。如需进一步细化某个子系统（如武举考试流程、门派修行剧情），我可以继续展开。
