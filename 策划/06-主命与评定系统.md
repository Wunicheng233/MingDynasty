# 06 - 主命与评定系统（太阁核心循环）

## 一、评定会框架（完全继承太阁4）

**评定会** 是整个游戏的节奏控制器，每两个月一次，就像太阁4的"每月评定"一样。

### 1.1 基本规则

- **周期**：奇数月1日自动召开
- **缺席惩罚**：玩家不在居城 → 扣除功勋，多次缺席可能被驱逐
- **流程**：主公发布主命 → 玩家选择 → 执行 → 复命结算功勋 → 身份晋升

### 1.2 两种游戏状态

| 状态 | 定位 | 核心目标 |
|------|------|---------|
| **家臣阶段** | 玩家接受主命，执行任务 | 积累功勋 → 晋升城主 |
| **君主阶段** | 玩家发布主命，给家臣 | 扩张势力 → 统一天下 |

## 四、功勋计算规则

- **基础功勋**：
  - 初级任务：50~100
  - 中级任务：100~200
  - 高级任务：200~400
  - 极难（出征）：500~1000

- **倍率**：超额完成目标 → 双倍功勋
- **加成**：
  - 与主公亲密度高 → 额外+50功勋
  - 重臣说好话 → 提升成功率
  - 合战大破敌军 → 额外功勋

## 五、君主阶段：从"接受"到"发布"

当玩家受封城主后，身份从"执行者"变为"发布者"：

### 5.1 城主权力

- **自行举办评定** → 给家臣发布主命
- **管理军资金** → 每项主命需要拨付预算
- **晋升家臣** → 家臣功勋累积到了可以晋升
- **接受大名主命** → 仍需服从上一级，抗命可能被讨伐

### 5.2 皇帝阶段：六部体系（明朝特色）

玩家成为皇帝后，主命体系与六部深度融合：

| 六部 | 对应主命类别 | 负责人 |
|------|------------|---------|
| **吏部** | 人才录用、考核升迁 | 吏部尚书 |
| **户部** | 征税、开垦、赈灾、盐政 | 户部尚书 |
| **礼部** | 科举、外交、祭祀 | 礼部尚书 |
| **兵部** | 征兵、训练、出征 | 兵部尚书 |
| **刑部** | 断案、修订律令 | 刑部尚书 |
| **工部** | 筑城、水利、皇陵 | 工部尚书 |

玩家从"事必躬亲"变为"运筹帷幄"，只决定国策，具体事务交给六部尚书去办。

### 5.3 考成法（明朝特色，张居正）

借鉴张居正"考成法"：
- 每条主命都有完成期限
- 考核官员政绩，合格升官，不合格降级
- 形成"立限责事，以事责人"的治理循环

## 六、明朝特色主命设计亮点

1. **科举与武举**：
   - 文官路线：秀才→举人→进士→进入六部
   - 武官路线：乡试→会试→殿试→武状元
   - 太阁没有这个体系，是洪武立志传独有特色

2. **开中法盐引**（商人路线专属）：
   - 向边关运粮 → 换盐引 → 到盐区取盐 → 内地贩卖
   - 明朝初年商人致富核心途径，太阁没有

3. **厂卫密探**（锦衣卫路线专属）：
   - 侦缉、缉捕、审讯官员
   - 明朝特务政治特色，太阁没有

4. **朝贡与海贸**：
   - 礼部管理各国朝贡
   - 市舶司管理海外贸易
