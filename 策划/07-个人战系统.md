# 07 - 个人战系统

## 一、基础机制（借鉴太阁4个人战）

### 1.1 触发场景

- 武馆修行切磋
- 向NPC请求切磋
- 遭遇山贼刺客
- 武举考试武艺环节
- 剧情单挑（如朱元璋单挑陈友谅部将）

### 1.2 核心属性

- **体力**：初始 = 武力 × 2 + 50，归零则战败
- **手牌数**：由武艺技能等级决定
  - 武艺Lv0：4张
  - 武艺Lv1：5张
  - 武艺Lv2：6张
  - 武艺Lv3：7张

### 1.3 回合流程

1. **抽牌阶段**：抽至手牌上限
2. **出牌阶段**：最多 1张红色特殊技卡 + 最多 3张数字卡
3. **判定阶段**：比较优先度，高者先攻
4. **结算阶段**：计算伤害，扣除体力，下一回合

### 1.4 牌组构成

| 类型 | 说明 | 获取 |
|------|------|------|
| **数字卡** | 1~9各一张，永远存在 | 初始拥有 |
| **特殊技卡（红）** | 主动招式，每场战斗随机抽取 | 修行习得 |
| **秘传卡（金）** | 名将专属，威力极大 | 特定条件习得 |

武艺等级越高，手牌越多，组合可能性越大。

## 七、优先度判定规则

优先度顺序：
```
秘传卡(S) > 特殊技(A) > 数字组合：三同张 > 三连张 > 二同张 > 无组合
```

同类型比较数字和：和大者优先。

## 八、伤害计算公式

```
最大体力 = 武力 × 2 + 50
基础伤害 = 武力 / 5 + 招式威力 × 10
实际伤害 = 基础伤害 × 组合系数 × 特殊技倍率 × 体力状态修正
```
