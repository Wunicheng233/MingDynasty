# 10 - 社交与关系网络

## 一、亲密度体系（借鉴太阁4）

亲密度是衡量玩家与其他角色关系的核心数值，范围 **-4（死敌）~ +5（生死相随）**：

| 心级 | 数值范围 | 关系 | 可解锁内容 |
|------|---------|------|-----------|
| -4 | -100 ~ -80 | 不共戴天 | 见面可能触发战斗 |
| -3 | -79 ~ -60 | 仇敌 | 拒绝一切互动 |
| -2 | -59 ~ -30 | 交恶 | 不可修行、不可劝诱 |
| -1 | -29 ~ -1 | 冷淡 | 可送礼，但效果减半 |
| 0 | 0 ~ 19 | 中立 | 初次见面 |
| 1 | 20 ~ 39 | 一面之缘 | 基础互动（送礼、交谈） |
| 2 | 40 ~ 59 | 略有交情 | **可拜师修行**、**可劝诱**加入 |
| 3 | 60 ~ 79 | 推心置腹 | **获得人物卡**、**可结义/求婚** |
| 4 | 80 ~ 94 | 莫逆之交 | 劝降成功率极高 |
| 5 | 95 ~ 100 | 生死相随 | 永不叛变（配偶/结义） |

### 亲密度变化规则

- **自然衰减**：每半年无互动 → -2（最低到0）
- **送礼**：+5~25，取决于礼物价值和对方喜好
- **茶会/宴饮**：成功+10~15，失败-2
- **切磋**：胜利+5（豪勇性格+10），失败+2
- **共同执行主命**：成功后+8 对同队所有人
- **合战救援**：救援成功+20
- **负面事件**：背叛、劝降失败 → 大幅降低

## 二、互动方式（每次消耗1天）

### 2.1 送礼

- 玩家从持有宝物卡中选一件赠予
- 对方接受 → 亲密度上升，宝物转移
- 对方婉拒 → 无变化，可重试
- 武将喜好武器兵法，文人喜好书画茶器，符合喜好加成50%

### 2.2 茶会 / 宴饮

- 茶会：5贯，邀请对方喝茶
- 宴饮：15贯，邀请对方喝酒
- 需要进行一次礼制/口才检定 → 成功提升亲密度

### 2.3 切磋武艺

- 请求与对方切磋 → 对方同意（亲密度≥1，性格非谨慎胆小）
- 进入个人战 → 胜负都加亲密度，胜利加更多
- 如果玩家已拥有对方人物卡，满足条件，**切磋胜利可能习得对方秘传卡**

### 2.4 交谈

子选项：
1. **打听情报** → 询问某城镇物价、某武将位置
2. **邀约同行** → 邀请一起修行/执行主命（亲密度≥2）
3. **请求指导** → 请求传授技能（对方技能等级高于你，亲密度≥2）→ 进入修行
4. **劝诱** → 说服对方加入你的势力（亲密度≥2，对方无主或不满主君）

### 2.5 结义 / 求婚

**条件**：亲密度≥4心，满足身份要求。

**婚姻**：
- 异性，玩家身份≥千户
- 需要聘礼（价值100贯宝物）
- 举办婚礼消耗3天，200贯
- 效果：亲密度锁定5心，配偶移居你的城镇自宅，每天互动可恢复15体力，偶尔赠礼

**结义**：
- 同性，身份差距不大
- 最多2个结义兄弟
- 举办仪式消耗1天，50贯
- 效果：亲密度锁定5心，永不叛变；合战同队 **士气+10**

## 三、明朝特色人际关系网

### 3.1 预设关系

- **淮西集团**：朱元璋同乡（徐达、汤和、李善长等）互相亲密度+20加成，玩家凤阳出身与之互动更容易
- **师生同年**：科举中第后，与考官、同年亲密度天然加成，官场互相提携
- **姻亲政治**：婚姻结成政治联盟，亲家之间亲密度加成
- **厂卫监视**：如果你得罪了锦衣卫，与某些官员亲密度提升时，锦衣卫会暗中破坏，送礼效果减半

### 3.2 NPC之间也有关系

不仅仅玩家与NPC有关系，NPC之间也存在亲密度网络，影响：
- 劝诱成功率（如果目标与现任主公亲密高，很难劝诱）
- 投票/建议时，朋友会帮你说话
- 主君对你的怀疑，可能有人帮你说话

## 四、与其他系统耦合

- **卡片系统**：亲密度3心 → 获得该角色人物卡
- **修行系统**：亲密度2心 → 可拜师修行
- **主命系统**：劝降主命成功率 = 口才×亲密度
- **合战系统**：结义兄弟同队 → 士气+10
- **事件系统**：很多事件需要特定亲密度才能触发

## 六、与太阁立志传4对比

| 太阁4 | 洪武立志传 |
|------|------------|
| 亲密度0-5心 | 亲密度-4~+5心，架构一样 |
| 送礼/茶会/切磋 | 送礼/茶会/切磋，架构一样 |
| 结婚/结义 | 结婚/结义，架构一样 |
| 忍者/海贼亲密度 | 明朝特色：淮西集团/师生/厂卫 |

核心社交机制完全继承太阁4，人际关系本土化改造为明朝特色。
