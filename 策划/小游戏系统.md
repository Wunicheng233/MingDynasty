## 《洪武立志传》小游戏系统完整设计

小游戏系统是《洪武立志传》的**成长检验器**与**趣味调味剂**。在《太阁立志传4》中，每一项技能的修行都以一个小游戏对应，从“运茶测温”到“连珠对战”，这些设计让枯燥的数值成长变得妙趣横生。你的文字游戏同样需要一个精巧的小游戏系统——它不能照搬图形界面游戏的交互方式，而需要找到“文字化表达”的独特魅力。


### 一、小游戏的设计困境与出路

**图形界面游戏的“原罪”**：原版《太阁立志传4》中的小游戏极度依赖图形交互——“运茶”需要看小人走格子、“算术”需要记忆5×5方盘的瞬间布局、“矿山”需要选择挖矿方向并比点数大小。这些玩法在纯文字环境下几乎无法直接呈现。

**文字游戏的“解法”**：当视觉无法承载，文字本身的魅力便浮现出来——描述、推理、选择、运气。文字游戏的小游戏，核心逻辑应该从“**眼力+反应**”转向“**策略+运气+文字演绎**”。我们可以用骰子掷点、选择路径、答题闯关、资源分配等纯文字可表达的机制，来还原每一种技能修行的“精神内核”。

**明朝元素的注入**：16项技能的小游戏不仅是一个玩法容器，更是一个绝佳的**历史文化输出窗口**。每一次修行，都是一次与明朝文化“亲密接触”的机会。


### 二、16项技能对应小游戏·完整本土化方案

以下为每一项技能设计一款适合文字游戏的小游戏，兼顾可玩性、明朝特色和与技能的关联度。

#### 2.1 步战 · 步军演武

**技能定位**：步兵统领能力，关联统率。

**小游戏名称**：步军演武

**核心机制**：简易战棋推演。双方各有一队步兵（基础兵力100），玩家在5回合内，每回合可选择三种战术指令之一，与敌军指令形成克制关系。

**战术克制链**（石头剪刀布式）：
- **【冲锋】** → 克制“坚守”（冲破防线，造成大量伤害）
- **【坚守】** → 克制“侧击”（稳住阵脚，反击侧翼敌军）
- **【侧击】** → 克制“冲锋”（绕开正面，从侧翼打击）

**胜负判定**：5回合内击溃敌军（兵力归零）为完胜；5回合后兵力高于敌军为小胜；否则失败。完胜可获得双倍技能经验。

**明朝典故**：元末乱世，朱元璋在郭子兴军中从步卒做起，屡立战功。此小游戏模拟的正是明初步兵方阵的基本战术思维。

**文字界面示例**：
```

═══════════════════════════════════════════
敌军步兵兵力：100    我军步兵兵力：100
回合：第2/5回

上一回合：你出【冲锋】，敌军出【坚守】→ 你冲破敌阵，敌兵力-35！

请选择本回合战术：
  1. 冲锋——克坚守，被侧击克

```


#### 2.2 骑战 · 赛马争先

**技能定位**：骑兵指挥与马术，关联武力/统率。

**小游戏名称**：赛马争先

**核心机制**：骰子驱动竞速。与对手赛马，从起点到终点共需累积15点进度。每回合双方掷骰子（1-6）决定前进步数。玩家在掷骰前可选择“策马方式”来影响结果。

**策马方式**：
- **【稳扎稳打】**：正常掷骰，无加成无惩罚。
- **【策马加鞭】**：掷骰结果+1（上限6），但若掷出1则马匹受惊，下回合无法行动。
- **【以身挡道】**：不增加进度，但对手本回合掷骰结果-2（下限1）。

**胜负判定**：先到达15点进度者胜出。完胜条件：领先对手至少3点进度取胜。骑战技能等级越高，【策马加鞭】触发受惊的概率越低（Lv1: 1点时触发，Lv2: 1-2点触发，Lv3: 仅1点触发）。

**明朝典故**：元末明初，骑兵是决定战场胜负的关键力量。常遇春以“单骑突阵”闻名，明军在与元军交战中缴获大量战马，逐步建立起自己的骑兵队伍。此小游戏模拟骑兵训练的竞速场景。

**文字界面示例**：

  1. 稳扎稳打——正常掷骰
  2. 策马加鞭——掷骰+1，但掷出1则下回合无法行动
  3. 以身挡道——不减进度，但对手下回合掷骰-2

#### 2.3 火器 · 神机校射

**技能定位**：火铳/火炮使用能力，关联智力/统率。

**小游戏名称**：神机校射

**核心机制**：风向+瞄准双因素射击。玩家需用火铳射击5个目标。每回合显示“风向”（顺风/逆风/侧风/无风）和“目标距离”（近/中/远）。玩家选择“瞄准部位”（头部/躯干/腿部）后掷骰，再加上风偏修正值，命中目标则得分。

**瞄准机制**：
- 目标为“近”时：头部需掷出5-6，躯干3-6，腿部1-6（必中）
- 目标为“中”时：头部需掷出6，躯干4-6，腿部2-6
- 目标为“远”时：头部不可命中，躯干需掷出5-6，腿部3-6
- 顺风：骰值+1；逆风：骰值-1；侧风：骰值无变化但腿部命中率降低

**胜负判定**：5次射击累计积分（头部3分，躯干2分，腿部1分），积分≥8为完胜，≥5为小胜，否则失败。火器技能等级越高，骰子面数从d6逐步提升至d8（Lv3），命中率自然提高。

**明朝典故**：明成祖朱棣组建神机营，是世界上最早的专业火器部队。沐英在云南平定叛乱时首创“三段击”轮射战术，比欧洲早数百年。此小游戏模拟神机营士兵的火铳射击训练。

#### 2.4 水战 · 楼船破浪

**技能定位**：水军指挥能力，关联统率。

**小游戏名称**：楼船破浪

**核心机制**：资源分配+随机事件。玩家指挥一艘战船穿越鄱阳湖，总航程15段。每段航行需要消耗“船工体力”或“水军士气”资源，并可能遭遇随机事件。初始资源各10点。

**每回合操作**：选择航行速度——
- **【全速前进】**：前进2段，消耗士气2点，可能遭遇暗礁（损失体力）
- **【中速巡航】**：前进1段，消耗体力1点，无事
- **【停船休整】**：不前进，恢复体力3点或士气3点

**随机事件**：每3段航程触发一次。如“遭遇敌船”需选择“接舷战”或“弓箭齐射”，消耗不同资源并掷骰决定成败；“风向突变”可能额外前进或后退等。

**胜负判定**：在任意资源归零前到达15段终点，为小胜；两种资源均≥5且到达终点，为完胜；资源归零则失败。

**明朝典故**：1363年鄱阳湖之战，朱元璋以少胜多击败陈友谅，是中国历史上最大规模的水战之一。此小游戏模拟水军战船在湖中航行的指挥调度。

#### 2.5 弓术 · 百步穿杨

**技能定位**：射箭能力，关联武力。

**小游戏名称**：百步穿杨

**核心机制**：拉力蓄力条。玩家有6支箭，每支箭需选择“拉力”（轻拉/中拉/满拉），拉力越大射程越远但精度越低。目标距离随机在“近/中/远”三档变化。玩家根据距离选择合适拉力，掷骰决定是否命中。

**拉力与命中规则**：
- 轻拉：近距必中，中距50%，远距0%
- 中拉：近距必中，中距70%，远距30%
- 满拉：近距50%，中距70%，远距60%

**胜负判定**：6箭累计命中数。6箭全中为完胜（极难），4-5中为小胜，否则失败。弓术技能等级越高，满拉的命中率逐步提升（Lv3时满拉远距命中率达80%）。

**明朝典故**：明朝武举考试中，射箭是必考项目。名将花云以善射闻名，守太平城时箭无虚发。此小游戏模拟射箭训练中的拉弓瞄准过程。

#### 2.6 武艺 · 武馆切磋

**技能定位**：个人战基础能力，关联武力。

**小游戏名称**：武馆切磋

**核心机制**：此小游戏直接**嵌入个人战系统**。玩家在武馆修行时，触发与师父或同门的切磋战。胜负判定、出牌规则完全沿用前文设计的个人战机制。

**修行效果**：切磋胜利可获得技能经验，连胜获得额外经验加成。武艺Lv1时手牌5张，Lv2时6张，Lv3时7张。

**明朝典故**：明朝民间武风盛行，少林寺、武当山等武术圣地与明王朝关系密切。朱元璋早年曾在凤阳皇觉寺习武，其所创“太祖长拳”流传后世。

**说明**：由于武艺修行直接调用个人战系统，无需额外设计独立小游戏，这也体现了系统的模块化复用。


#### 2.7 兵法 · 排兵布阵

**技能定位**：军事谋略，关联智力。

**小游戏名称**：排兵布阵

**核心机制**：阵型相克问答+布阵推演。玩家面对一个敌军阵型，需在3个选项中选出克制该阵型的我方阵型。连续回答正确3题后，进入一场简化的“军学双六”推演——从起点到终点共8格，每回合掷骰前进，途中可能遭遇“敌袭”（损失步数）或“识破敌军”（额外前进）。

**阵型相克知识**（基于前文合战系统设计）：
- 鱼鳞克雁行，雁行克锋矢，锋矢克衡轭，衡轭克方圆，方圆克长蛇，长蛇克偃月，偃月克鹤翼，鹤翼克鱼鳞

**胜负判定**：8回合内到达终点为小胜；答对所有问题且6回合内到达终点为完胜；超时则失败。兵法技能等级越高，推演途中遭遇负面事件的概率越低。

**明朝典故**：徐达、刘伯温等明朝开国元勋皆精通兵法。徐达善用“车轮战术”，刘伯温以奇谋著称。此小游戏模拟军中幕僚推演战局的过程。

#### 2.8 农政 · 劝课农桑

**技能定位**：农业开垦能力，关联内政。

**小游戏名称**：劝课农桑

**核心机制**：资源分配。玩家有10块待垦荒地，初始资金50贯。每块地需投入“人力”和“牛力”两种资源。每回合可选择：
- **【开垦荒地】**：投入资金开垦一块地
- **【修建水利】**：花费10贯，提升所有已垦地的产出
- **【招募佃农】**：花费5贯，提升人力

**随机事件**：每开垦3块地触发一次——“旱灾”、“蝗灾”、“丰收”等，影响产出或消耗资金。

**胜负判定**：将10块地全部开垦完毕，且剩余资金≥20贯为完胜；全部开垦但资金<20贯为小胜；资金耗尽则失败。农政技能等级越高，初始资金越多，随机事件中受灾概率越低。

**明朝典故**：朱元璋出身农家，深知民间疾苦。立国后推行“高筑墙、广积粮”战略，大力开垦荒地、兴修水利。此小游戏模拟地方官员劝课农桑的施政过程。

#### 2.9 工政 · 筑城考工

**技能定位**：工程建设能力，关联内政。

**小游戏名称**：筑城考工

**核心机制**：目标重量趋近。玩家需用三种石块（大块500贯、中块250贯、小块50贯）垒砌城墙，使总重量尽可能接近目标重量（如1700贯）。每回合选择添加一种石块，累计到≥目标重量即结束。

**策略要点**：先用大块接近目标，再用中小块微调。内政越高，最后凿子修正时能敲掉的重量越精确（每次90-100贯左右）。

#### 2.10 商政 · 珠算商贾

**技能定位**：商业算术能力，关联内政。

**小游戏名称**：珠算商贾

**核心机制**：快速心算。玩家扮演商贾，连续回答6道算术题。题目类型包括：单价×数量、折扣计算、钱币换算（贯/文/两）、利润计算等。每道题限时回答（文字游戏中可以用“余3秒”等倒计时提示制造紧张感）。

**题目示例**：
- “上好丝绸一匹售价3贯200文，客官买4匹，需付多少钱？”（答案：12贯800文）
- “盐引一张可支盐300斤，若每斤盐售价80文，一张盐引的盐值多少钱？”（答案：24贯）
- “原本利润45两，因漕运延误被扣三成，还剩多少两？”（答案：31.5两）

**胜负判定**：6题全部答对为完胜（获得双倍经验），答对4-5题为小胜，否则失败。商政技能等级越高，题目涉及的运算越复杂但限时越长。

**明朝典故**：明朝中后期商品经济发达，晋商、徽商崛起。“开中法”盐引贸易是商人致富的重要途径。此小游戏模拟商贾在交易中快速算账的场景。

#### 2.11 律政 · 明律断案

**技能定位**：法律知识，关联智力/魅力。

**小游戏名称**：明律断案

**核心机制**：证词矛盾分析。玩家作为一名地方官审理案件，会看到原告、被告、证人三方的简短证词。证词中存在1-2处相互矛盾的地方，玩家需指出哪两句话存在矛盾，并从选项中选出正确的判决依据（出自《大明律》哪一条）。

**案件示例**——“田产纠纷案”：
- 原告（老农）：“这块田是我祖上传下来的，种了三十年，去年借给被告耕种，今年他不还了！”
- 被告（富户）：“这田是我三年前从原告手中买下的，有地契为证。”

判决依据：《大明律·户律·田宅》“盗卖田宅”——凡盗卖他人田宅者，田产归还原主，盗卖者杖八十。

**胜负判定**：正确指出矛盾点得1星，正确选择律条得1星，两者全对（2星）为完胜；仅对一项为小胜；全错失败。律政技能等级越高，证词篇幅越长、矛盾越隐蔽，但会获得更多提示。

**明朝典故**：《大明律》是中国法制史上具有重要地位的成文法典，朱元璋亲自主持修订。此小游戏模拟地方官依律断案的司法过程。

#### 2.12 口才 · 游说纵横

**技能定位**：辩舌游说能力，关联魅力/外交。

**小游戏名称**：游说纵横

**核心机制**：情绪值博弈。玩家游说一名目标角色（如敌将、商人、官员）。对方有一个隐藏的“情绪值”（初始0），玩家每回合可选择三种话术之一，影响对方情绪，同时观察对方反馈的文字描述来判断其当前情绪状态。

- **【动之以情】**：打感情牌，适合对方情绪低落时（情绪值+15~20），对方得意时可能被视为软弱（情绪值-10）
- **【诱之以利】**：许诺好处，适合对方摇摆时（情绪值+20~25），对方警觉时可能起疑心（情绪值-15）

- 对方“目光闪烁，似在权衡”——摇摆状态，适合诱之以利
- 对方“眉头紧锁，冷哼一声”——警觉状态，诱之以利可能起反效果
- 对方“勃然大怒”——激动状态，晓之以理可能激怒

#### 2.14 茶道 · 茶香论道

**技能定位**：茶道修养，关联魅力。

**小游戏名称**：茶香论道

**核心机制**：温度控制。原版《太阁立志传4》的运茶游戏极其考验耐心——让小人在棋盘上行走，每走一步温度降一格，还需应对各种方向格和地雷，被玩家公认为最难的小游戏之一。文字游戏化改造的核心思路是：去掉空间移动的复杂性，保留“温度控制”这一精神内核。

**游戏流程**：
- 初始茶温为100度，目标茶温为65度
- 每回合玩家可选择一种“煮茶手法”，不同的手法降低不同的温度值，但每次降低的数值有一定随机浮动（±3度）
- 玩家需在5-8回合内将茶温精确降至65±2度范围内

**煮茶手法选项**：
- **【扇风降温】**：温度-8~12度（随机）
- **【注入凉水】**：温度-15~20度（随机）
- **【静置片刻】**：温度-3~5度（随机）
- **【文火微温】**：温度+3~6度（随机）

**策略要点**：先用大幅度降温手法快速接近目标范围，再用微调手法（静置或文火）逼近目标。若温度低于60度或高于70度持续2回合，茶汤报废失败。

**胜负判定**：在规定回合内将茶温精确控制在65±2度范围内，为完胜；在±5度范围内为小胜；超出范围或超时则失败。茶道技能等级越高，温度浮动范围越小（更容易精确控制），可用的手法越多。

**明朝典故**：明代茶文化盛行，朱元璋“废团改散”，推动散茶瀹泡法普及。朱权著《茶谱》，对明代茶道影响深远。此小游戏模拟文人雅士点茶品茗的风雅场景。

### 三、小游戏系统架构

#### 3.1 小游戏与修行系统的耦合

小游戏是修行系统的核心玩法载体，两者关系如下：

```
玩家在城镇找到对应师父（需亲密度≥2心）
        ↓
选择“师事修行”→ 进入对应技能的小游戏
        ↓
根据小游戏结果评定等级（完胜/小胜/失败）
        ↓
        ├─ 完胜：技能经验+2，可能习得对应战术卡/武技卡
        ├─ 小胜：技能经验+1
        └─ 失败：无经验
        ↓
经验累积满 → 技能等级+1 → 解锁新的阵型/战术卡/手牌数等
```

#### 3.4 跳过机制设计

参考《太阁立志传4》允许玩家在PS2及之后版本中跳过小游戏的选项，你的游戏也应提供类似功能：

- 当玩家某项技能达到Lv3后，再次修行该技能时可选择“跳过小游戏”
- 跳过时只能获得小胜对应的经验（+1），无法获得完胜的额外经验和新卡片
- 这是为不喜欢反复玩小游戏的玩家提供的便利，同时保留了“完胜才能解锁卡片”的激励机制


### 四、小游戏系统与各模块的耦合关系

小游戏系统并非孤立存在，它与游戏的多个核心系统紧密联动：

| 关联系统 | 耦合方式 | 数据流向 |
|---------|---------|---------|
| **修行系统** | 小游戏是修行系统的玩法载体，游戏结果决定技能经验获得 | 小游戏 → 技能等级 |
| **卡片系统** | 小游戏首次“完胜”可解锁对应技能等级的战术卡/武技卡 | 小游戏 → 卡片收集 |
| **主命系统** | 部分政务类主命（如“征收赋税”）会调用商政小游戏；“断案”调用律政小游戏 | 主命 → 小游戏 |
| **属性成长** | 技能等级提升后，关联属性（如内政、魅力）也会获得少量成长 | 技能等级 → 属性值 |
| **合战系统** | 步战/骑战/火器等技能等级越高，合战中可用阵型和战术卡越多 | 技能等级 → 合战能力 |
| **个人战系统** | 武艺/密探/医术技能等级决定手牌数和可用武技卡 | 技能等级 → 个人战能力 |
| **经济系统** | 商政小游戏的部分题目涉及钱币换算，间接教学经济系统 | 小游戏 → 经济知识 |
| **时间系统** | 每次修行小游戏消耗5天时间，无论胜负 | 小游戏 → 时间推进 |


### 五、实现建议（文字游戏开发）

**1. 小游戏引擎架构**

建议采用“游戏状态机”模式，每个小游戏实现统一的接口：

```python
class MinigameBase:
    def start(self):        # 初始化游戏，生成题目/布局
    def step(self, choice): # 处理玩家输入，返回结果和是否结束
    def get_state(self):    # 返回当前游戏状态供界面渲染
    def evaluate(self):     # 判定胜负等级（完胜/小胜/失败）
```

**2. 数据驱动设计**

小游戏的题目、参数等应存储在JSON配置文件中，便于后续扩展和调整平衡性：

```json
{
  "skill": "商政",
  "minigame": "珠算商贾",
  "levels": [
    {"level": 1, "question_count": 6, "time_limit": 15, "difficulty": "easy"},
    {"level": 2, "question_count": 6, "time_limit": 12, "difficulty": "medium"},
    {"level": 3, "question_count": 6, "time_limit": 10, "difficulty": "hard"}
  ],
  "questions": [
    {"text": "丝绸一匹3贯200文，买4匹需付多少", "answer": "12贯800文", "type": "multiplication"},
    ...
  ]
}
```

**3. 界面渲染**

文字游戏的小游戏界面可采用“状态描述+菜单选项”的模式：

```
═══════════════════════════════════════════
        [技能名称]修行 · [小游戏名称]
═══════════════════════════════════════════
[当前状态描述行1]
[当前状态描述行2]
...
═══════════════════════════════════════════
请选择：
  1. [选项1描述]
  2. [选项2描述]
  3. [选项3描述]
...
═══════════════════════════════════════════
```

**4. 随机性与可重玩性**

每个小游戏都应有足够的随机性，避免每次修行的体验完全相同：
- 步战：敌军AI的出牌策略有一定随机权重
- 火器/弓术：目标距离、风向每局随机
- 商政：从题库中随机抽取6题
- 律政：从案件库中随机抽取
- 密探：节点选项的排列每局随机

**5. 存档兼容**

小游戏本身不产生存档数据，只有修行结果（技能经验变化、卡片获取）会写入存档。因此无需为小游戏设计独立的存档结构，只需在修行系统结算时更新角色数据。


### 六、总结

《洪武立志传》的小游戏系统，在继承《太阁立志传4》“每项技能对应一个小游戏”经典框架的基础上，完成了三个层面的本土化改造：

1. **玩法降维**：将原版依赖图形交互的玩法（运茶走格子、算术翻牌等），改造为纯文字可表达的“策略+运气+文字演绎”机制。骰子掷点、选项路径、顺序记忆、证词推理——这些核心机制天然适合文字游戏，且不失策略深度。

2. **文化注入**：每一项小游戏都与明朝历史深度融合——火器关联神机营、水战关联鄱阳湖、律政关联《大明律》、茶道关联废团改散、农政关联高筑墙广积粮。修行不仅是数值成长，更是一次次文化沉浸。

3. **系统联动**：小游戏不是孤立的“玩票”内容，而是修行系统的核心引擎、卡片系统的主要解锁途径、合战/个人战能力的基础支撑。完胜解锁卡片的设计，为玩家提供了“反复修行追求完美”的动力。

这套小游戏系统既保留了《太阁立志传》系列“修行有趣”的精髓，又通过文字化的巧妙改编，让单人开发成为可能。
