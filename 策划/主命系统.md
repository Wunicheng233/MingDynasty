## 《洪武立志传》主命系统完整机制

主命系统是整个游戏的**核心推进器**——玩家的一切成长、晋升、剧情推进，都围绕“接受主命→执行主命→结算功勋→身份晋升→解锁更多主命”这一循环展开。而这一系统在玩家身份变化后，玩法逻辑将发生根本性转变：**从“听命于人”变为“发号施令”**。

以下分别阐述两种状态下的完整机制。


### 一、通用基础：评定会框架

不论玩家处于何种身份，游戏的节奏都由**评定会**来统御。

**评定会周期**：
- 每隔**两个月**召开一次评定会（固定于奇数月一日，如1月1日、3月1日……）
- 评定会当日，玩家必须身处主公（或自己）的居城，否则视为**缺席**
- 缺席评定会将扣除功勋，多次缺席可能引发严重后果（降职甚至被驱逐）

**评定会流程**：
1. **主公发布主命**（或玩家自主选择主命方向）
2. **玩家选择接受主命**（可主动请缨，也可能被主公强行指派）
3. **执行阶段**（持续1~60天不等，视任务复杂度而定）
4. **复命结算**（任务完成后回到主公处交差，获得功勋与奖励）

**主命执行期间的限制**：
- 身上带有主命时，头像下方会显示主命内容及当前完成进度
- 主命期间仍可自由移动、修行、社交，但需注意时间限制
- 若未在规定时间内完成，将被视为任务失败，扣除功勋并降低主公亲密度


### 二、状态一：身为家臣（听命于人）

这是玩家最初的身份状态。从一介草民加入红巾军/明军后，便进入这套“听命于人”的主命体系。

| 小旗 | 200 | 300 | 初级军务 | 从七品 |
| 总旗 | 600 | 500 | 初/中级军务 | 正七品 |

**进阶国主条件**：功勋达到6000以上，拥有一定数量的属下，回到居城后可能被直接晋升为**国主**，该国家所有直辖城都会划归玩家管辖。

#### 2.3 功勋计算规则

**基础功勋**：每项主命有预设的基础功勋值，根据任务难度和完成效果浮动：
- 初级任务：50~100功勋
- 中级任务：100~200功勋
- 高级任务：200~400功勋
- 极难任务（出征攻城等）：500~1000功勋

**倍率机制**：完成任务达到2倍或超过2倍预期效果，可获得**双倍功勋**。

**加成因素**：
- 与主公亲密度越高，主公越可能采纳你的建议，额外获得50功勋
- 与家中重臣亲密度高，他们会帮你说话，提高任务成功率
- 合战中立下大功（击破敌将、破城等）可额外获得大量功勋

#### 2.4 主公建议机制

在评定会上，玩家可向主公提出战略性建议，例如：
- “应优先攻打陈友谅”（建议出征方向）
- “应与张士诚暂时同盟”（外交方向）
- “应加强水军建设”（军备方向）

**建议采纳率**由以下因素决定：
- 与主公亲密度（核心因素）
- 玩家身份等级（身份越高，话语权越大）
- 与家中重臣的亲密度（他们会帮腔）

建议被采纳后，不仅能额外获得功勋，还能**按自己的意志影响势力的战略方向**，对后续剧情发展产生深远影响。


### 三、状态二：身为君主（发号施令）

当玩家通过功勋晋升为**城主/国主**，或通过谋反/继承成为**大名**（对应明朝的吴王/皇帝），主命系统的身份从“执行者”翻转为“发布者”。

#### 3.1 城主的职责与权力

成为城主后，玩家需要**自行举办评定**，给予家臣主命，并承担经营内政与充实军备的责任。

**城主可用主命**：可以向属下发布的命令类型与上述“家臣主命”基本一致，按类别可分为：
- **内政类**：筹措军资金、买卖兵粮、开发新田、开发矿山、修补城墙等
- **军备类**：征兵、训练、购入军马、购入火器
- **调略类**：人才调查、城调查、破坏（城/流言/放火）、劝诱浪人
- **外交类**：同盟、破弃、支配、宣战、和睦、亲善、向朝廷献金

**军资金管理**：城主的军资金是向属下发布主命的“资源”。每向属下发布一项主命，都需要从城中军资金中拨出预算。擅长“算术”及“口才”的属下执行“筹措军资金”主命，是不错的增收选择。

**属下晋升**：若想给予属下更高级的主命，则需提升属下的身份。属下的功勋积累到一定程度后，玩家可为他们举办晋升仪式。

**来自大名的命令**：身为城主/国主，仍需接受来自大名的“攻略○○○”主命，即攻打指定敌城。完成任务后可获得大量功勋，反之若抗命不遵，可能被大名猜忌甚至讨伐。

#### 3.2 国主的额外权力

当玩家成为国主后，可支配多座城池，并在所辖城之间自由**输送物资**（如将粮草从后方运往前线）。同时可以向忍者众、水军众请求援军，前提是通过“外交·亲善”提升友好度，再以“外交·支配”使其从属。

#### 3.3 成为大名的途径

玩家最终可以从国主晋升为独立的大名（对应明朝的皇帝/吴王）：

- **谋反**：身为城主或国主时，在评定期间选择“谋反”，宣布脱离主家独立。但此举会使与背叛大名的关系降至最低，需事先充分做好军备，否则非常危险
- **继承**：若大名因某些原因死亡（如战死、病故、剧情），且玩家满足条件（如为家族重臣、功勋足够等），便有机会成为继承人
- **特殊剧情**：朱元璋第一章主线中，郭子兴病故后，朱元璋继承其势力，正式成为独立势力主

#### 3.4 大明皇帝/吴王状态下的评定会

成为真正的君主后，评定会的内容将发生根本变化：

**评定会新增指令**：

| 指令 | 说明 | 明朝特色 |
|------|------|---------|
| 发布主命 | 向家臣分派任务（同城主） | — |
| 战略决策 | 决定势力整体战略方向 | 类似朝会，制定国策 |
| 开科取士 | 举办科举，选拔文官 | **明朝独有** |
| 武举选将 | 举办武举，选拔武官 | **明朝独有** |
| 朝贡外交 | 与他势力进行高层外交 | 明初对元/诸势力 |
| 册封官员 | 授予属下官位（正一品至从九品） | 明朝官制体系 |
| 颁布法令 | 推行政策法令（如一条鞭法前身） | 内政改革 |
| 祭祀天地 | 提升朝廷贡献度与名声 | 天子祭天 |

**战略决策层级**：作为君主，玩家不再接受他人的“主命”，而是自行制定“国策”——即势力的整体战略方向。国策决定了下一次评定会向家臣发布的战略目标，例如“攻略陈友谅”、“发展应天府经济”、“加强水军”、“北防元军”等。

**六部主命体系**（明朝特色）：成为皇帝后，主命系统与六部官制深度融合：

| 六部 | 对应主命类别 | 负责官员 |
|------|------------|---------|
| **吏部** | 人才录用、官员考核、升迁贬谪 | 吏部尚书 |
| **户部** | 征税、开垦、赈灾、盐政、漕运 | 户部尚书 |
| **礼部** | 科举、外交、祭祀、朝贡 | 礼部尚书 |
| **兵部** | 征兵、训练、出征、军备 | 兵部尚书 |
| **刑部** | 断案、缉捕、律令修订 | 刑部尚书 |
| **工部** | 筑城、水利、皇陵、火器制造 | 工部尚书 |

玩家可向六部尚书分派“部务主命”，由尚书再向下属官员分发。玩家直接过问的是“国家大政”，而非琐碎的日常任务。

**“考成法”考核机制**（明朝特色）：张居正“考成法”的核心是“立限责事，以事责人，务责实效”。在游戏中可设计为：每条发布的国策/主命都有完成期限，超期未完成则相应官员考核降级。官员政绩优异者可晋升，碌碌无为者被裁撤，形成“月有考，岁有稽”的治理循环。


### 四、两种状态的核心差异对比

| 维度 | 家臣状态（听命于人） | 君主状态（发号施令） |
|------|-------------------|-------------------|
| **评定会定位** | 接受任务，积累功勋 | 发布任务，管理家臣 |
| **核心目标** | 升职→成为城主 | 扩张势力→统一天下 |
| **资源管理** | 个人俸禄与资产 | 军资金、城池、家臣 |
| **决策自主权** | 有限（可建议，但最终由主公决定） | 完全自主 |
| **主命执行者** | 玩家自己 | 属下家臣 |
| **任务种类** | 个人能力范围内的小型任务 | 国家大政，六部协同 |
| **功勋来源** | 完成主公指派的主命 | 势力扩张、国策成效 |
| **失败后果** | 扣功勋、降低主公亲密度 | 战略受挫、国力下降 |
| **进阶目标** | 功勋达标→晋升城主 | 攻灭他势力→统一天下 |


### 五、明朝特色主命系统设计亮点

**1. 科举与武举主命**
文官路线的核心晋升方式：玩家可参加科举考试，从秀才→举人→进士，最终进入六部任职。武官路线则通过武举选拔，比试射箭、骑术、兵法。这是《太阁》日本背景所不具备的独特玩法。

**2. 六部主命分流**
成为皇帝后，任务不再琐碎，而是按六部职能分流：
- 想打陈友谅？给兵部下令
- 想修南京城墙？给工部下令
- 想征收粮草？给户部下令
- 想招揽刘伯温？给吏部下令

玩家从“事必躬亲”转变为“运筹帷幄”。

**3. 考成法考核**
借鉴张居正“考成法”，家臣执行主命有明确的完成期限和考核标准。政绩优异的官员晋升，尸位素餐者被裁撤，形成动态的人事管理体系。

**4. 开中法盐引任务**
商人路线专属：向边关运送军粮换取盐引，再贩盐获利。这是明朝初期商人致富的核心途径，也与户部的盐政管理相关联。

**5. 厂卫密探主命**
锦衣卫路线专属：接受“侦缉”、“缉捕”、“审讯”等特殊主命，可对官员进行暗中监视和直接逮捕，是明朝特务政治的特色体现。

### 七、总结

《洪武立志传》的主命系统以“评定会循环”为节奏控制器，以“身份跃迁”为核心驱动力。从红巾军小卒到五军都督，再到受封城主、成为一方势力主，最终登基称帝——这条晋升之路与主命系统深度融合：

- **家臣阶段**：通过执行主命积累功勋，学习技能，收集卡片，同时可通过“主公建议”影响势力方向
- **君主阶段**：从“被管”变为“管人”，主命系统从个人任务扩展为六部协同的国家治理体系

这种**“从士兵到将军，从将军到皇帝”**的视角切换，正是《太阁立志传》系列的核心魅力所在，而明朝六部官制、科举武举、考成法等元素的加入，则让《洪武立志传》拥有了独特的历史质感。
