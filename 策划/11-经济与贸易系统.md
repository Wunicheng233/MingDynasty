# 11 - 经济与贸易系统

## 一、货币体系（明朝本土化）

| 单位 | 换算 | 主要用途 |
|------|------|---------|
| **文** | 基本单位 | 小额交易 |
| **贯** | 1贯 = 1000文 | 日常买卖、军资金、俸禄 |
| **两（银）** | 1两 ≈ 1.5贯 | 贵重宝物、大宗贸易、盐引 |

游戏内部统一用**文**存储，显示时自动换算为贯/两。

## 二、物价体系

### 2.1 物价波动影响因素

- **丰歉**：丰收年粮价下跌，灾年上涨
- **战乱**：交战城镇所有物价+20%
- **季节**：秋收后粮价最低，春荒最高
- **商路阻断**：围城或海禁 → 特产价格飙升
- **投资等级**：城镇投资越高，物价越稳定

玩家**商政**技能等级越高，交易界面显示的"建议买入/卖出"越准确。

### 2.2 基础交易（太阁式市集）

每个城镇市集都可以买卖商品，低买高卖赚取差价——这是商人路线初期的核心玩法：

```
═══════════════════════════════════════════
            应天府 · 市集
═══════════════════════════════════════════
持有金钱：320贯

商品 库存 买入价 卖出价
大米 2000石 1.2贯 0.8贯
丝绸 500匹 8贯 6贯
茶叶 800斤 3贯 2贯
盐 300引 15贯 12贯

1. 买入 2. 卖出 3. 查看其他城镇物价 4. 离开
═══════════════════════════════════════════
```

### 2.3 城镇投资

- 玩家可以投资城镇，提升**投资等级**
- 投资消耗金钱，每100贯 → 投资经验+1 → 繁荣度+1%
- 效果：解锁高级特产，降低物价波动，增加税收
- 城主/君主收税时，税收 = 繁荣度 × 税率

## 三、明朝特色商贸机制

### 3.1 开中法·盐引贸易（商人路线核心）

**历史背景**：明初为解决边关军粮，允许商人向边关运粮，换取**盐引**，凭盐引到两淮/长芦盐场支取食盐贩卖，是明初商人致富第一途径。

**游戏实现**：
1. 在边关城镇（大同、宣府）**接取运粮任务**
2. 购买粮食运输到边关 → 换取对应数量**盐引**（宝物卡）
3. 持盐引前往产盐区（扬州、淮安）**盐运司** → 兑换食盐
4. 将食盐运往内地高价出售 → 暴利
5  商政技能越高，每张盐引兑换的食盐越多；口才越好，卖出价格越高

**风险**：无盐引私自贩盐是**私盐**，被查获没收货物+罚款+降亲密度。

### 3.2 海禁与走私

**背景**：明初实行严厉海禁，只允许官方朝贡贸易，但沿海走私猖獗。

- **合法海商**：需要获得**市舶司**许可（身份指挥使以上），交关税，利润正常，风险低
- **走私**：不需要许可，风险高但**利润是合法的3~5倍**
  - 出海需要通过密探检定 → 成功安全抵达，利润满满
  - 失败被缉拿 → 没收全部货物，降低与该势力关系，可能罚款

### 3.3 宝钞与通货膨胀

**背景**：明初发行**大明宝钞**，但很快贬值。

- 游戏中，随着年份推移（洪武中后期），宝钞购买力**逐年下降**
- 玩家如果持有大量宝钞，需要及时兑换为白银/实物保值
- 这增加了经济运营的策略深度

### 3.4 商帮会馆

在扬州、苏州、太原等大城市有晋商、徽商、闽商商帮会馆：

- 玩家可以**加入商帮**（需要商政Lv2+会费）
- 加入后 benefits：
  1. 可查询**全国各地精准物价情报**
  2. 可承接**商帮专属主命**（运送漕粮、承办边饷）
  3. 可**借贷**应急资金
  4. 提升亲密度可以晋升 → 最终成为**会首**，掌控整个商帮网络

## 四、身份与经济权限

| 身份 | 可从事活动 |
|------|-----------|
| 平民/浪人 | 市集倒卖、打工 |
| 百户以下 | 领取俸禄（微薄）、执行筹措主命 |
| 千户以上 | 领地税收、参与开中法 |
| 城主/国主 | 管理城池金库、调整税率、投资城镇 |
| 君主 | 全国财政、决定赋税政策、发行宝钞 |

### 税收

- **家臣俸禄**：每两月评定发放，百户≈20贯，五军都督≈300贯
- **城主税收**：每季度从所辖城镇征收，总额 = 繁荣度 × 税率
  - 税率高 → 税收多，但民忠下降，可能暴动
- **君主财政**：全国税收总和 + 朝贡收入 - 官员俸禄 - 军费
  - 长期赤字 → 军队士气下降，可能叛乱

## 五、宝物系统

**宝物卡**是经济+社交的联结点：

- 宝物可以**装备** → 提供属性加成（如玄铁甲+15最大体力）
- 宝物可以**送礼** → 提升亲密度
- 宝物可以**出售** → 换钱

**宝物分类**：
- 武器：增加武力
- 防具：增加体力上限
- 书画/茶器：送礼佳品，文人喜好
- 丹药：恢复体力，但是加丹毒

## 六、与太阁立志传4对比

| 太阁4 | 洪武立志传 |
|------|------------|
| 町之间买低卖高 | 城镇之间买低卖高，架构一样 |
| 投资町增加规模 | 投资城镇增加繁荣，架构一样 |
| 大名进贡 | 开中法盐引（明朝特色）|
| 海贼贸易 | 海禁走私（明朝特色）|
| 忍者里/海贼砦 | 商帮会馆（明朝特色）|
| 武器购买 | 宝物卡系统，一样 |

核心贸易机制完全继承太阁4，加入了多个明朝特有的经济制度。
