在《洪武立志传》的世界里，地图就是你写下的传奇，而城镇就是承载这些故事的舞台。要让这个舞台活起来，每个城镇都需要有自己的呼吸和脉搏。

我们可以借鉴《太阁立志传5》的城市设计逻辑，其核心在于**为不同人生道路的玩家提供差异化的成长舞台**：从武士起家的政治军事中心，到商人发迹的商业枢纽，再到剑豪、海贼等特殊职业的专属据点，每个城市都有独特的设施配置和战略价值。

基于这个理念，我将游戏的城镇体系分成了四个等级，它们共同构成了一个立体的世界。

### 🗺️ 明朝城镇等级与规模

| 等级 | 规模（人口/兵力） | 特点 | 代表城镇 |
| :--- | :--- | :--- | :--- |
| **都城级** | 30,000人 / 6,000兵 | 政治、文化中心，设施最全，唯一拥有皇宫/国子监等顶级设施 | 应天府、大都、凤阳 |
| **省会级** | 20,000人 / 4,000兵 | 区域政治、经济中心，拥有布政使司、大型市集等高级设施 | 苏州府、杭州府、武昌府等 |
| **府城级** | 12,000人 / 2,500兵 | 地区性中心城市，设施较为完善，是玩家的主要活动区域 | 扬州府、镇江府、庐州府等 |
| **县城级** | 6,000人 / 1,000兵 | 基础物资补给点，设施较少，适合新手熟悉游戏 | 句容县、丹徒县等 |

### 🏙️ 城镇设施一览

每个城镇的“性格”由其拥有的设施决定。你可以在这里招募士兵、修炼武艺、买卖商品，或是触发各种奇遇。

| 设施类型 | 设施名称 | 主要功能 | 关联技能/玩法 |
| :--- | :--- | :--- | :--- |
| **军事设施** | **校场** | 招募与训练步兵、骑兵 | 步战、骑战、统率 |
| | **弓兵训练场** | 招募与训练弓兵 | 弓术 |
| | **神机营** | 招募与训练火器兵 | 火器 |
| | **水寨** | 招募与训练水军 | 水战 |
| | **兵器司** | 购买/升级个人武器、盔甲 | 个人战、经济 |
| **行政设施** | **官衙/王府** | 进行评定、接受或发布主命 | 主命系统、身份晋升 |
| | **工部司** | 进行筑城、水利、火器制造等修行 | 工政、火器 |
| | **户部司** | 进行开垦、算术、交易等修行 | 农政、商政 |
| | **国子监** | 学习经义、准备科举（仅都城/省会） | 文墨、礼制、文官路线 |
| | **刑部司/按察司** | 学习律法、断案修行 | 律政 |
| **经济设施** | **市集** | 买卖基础商品（粮、盐、布等） | 商政、经济 |
| | **商会会馆** | 大宗贸易、投资、获取商机（仅商业城镇） | 商政、商人路线 |
| | **钱庄** | 存取金钱、汇兑 | 经济、便利 |
| **社交设施** | **武馆** | 学习基础武艺、步战、骑战 | 武艺、个人战 |
| | **茶楼/酒馆** | 宴请、结交武将、打听情报 | 口才、社交 |
| | **寺庙/道观** | 学习医术、茶道，或触发特殊事件 | 医术、茶道、礼制 |
| | **自宅** | 休息、存档、与家人互动 | 存档、社交 |
| **特殊设施** | **锦衣卫所** | 接受密探任务、学习潜入技能（仅部分城镇） | 密探、厂卫路线 |
| | **书院** | 学习兵法、纵横术 | 兵法、口才 |
| | **名胜古迹** | 游览、获得名所卡 | 收集、探索 |

### 👥 主要人物初始位置分布

至正十三年（1353年），天下大乱，群雄并起。各方势力的核心人物已齐聚在他们的首府，为各自的野心谋划着：

| 势力 | 首府 | 初始核心人物 |
| :--- | :--- | :--- |
| **朱元璋军** | 濠州 | **朱元璋**（投身义军）、**徐达**、**汤和**、**李善长**、**郭子兴**（君主）等淮西旧部 |
| **元廷** | 大都 | **元顺帝**、**王保保**（扩廓帖木儿）等元朝核心将领 |
| **陈友谅军** | 江州/武昌 | **陈友谅**及其麾下猛将**张定边**等 |
| **张士诚军** | 平江（苏州） | **张士诚**，占据江南富庶之地，麾下聚集了一批文士和商人 |
| **方国珍军** | 庆元（宁波） | **方国珍**，控制浙东沿海，拥有强大的水军 |

*   **其他在野名士**：如**刘基**（刘伯温）、**宋濂**等浙东文人，则分散在处州、金华等地的书院或山中，等待玩家去发现与招揽。

### 🚶 人物动态移动机制

这个世界并非静止的画卷，人物也会像你一样，为了各自的目的一直在路上，形成动态的社交网络。他们的动向大致有三种：

*   **指令性移动（最常见）**：
    *   **执行主命**：当你接受任务前往某地时，你可能会在路上或目的地偶遇同样在执行任务的同僚或对手。
    *   **军事调动**：战争期间，将领会率军出征，你可以在地图上看到军队的移动，并可能发生遭遇战。
    *   **资源输送**：将领或商人会往返于城市之间进行物资输送，你可以选择拦截或保护。

*   **规律性移动（可预测）**：
    *   **述职/朝贡**：地方官员会定期前往首府参加评定或述职；势力君主或使者会定期前往上级势力进行外交朝贡。
    *   **修行/云游**：一些剑豪、医师、僧侣等会定期在几个固定地点（如武馆、寺庙）之间云游，传授技艺。

*   **事件驱动性移动（最具故事性）**：
    *   **历史剧情**：随着主线推进，人物的命运和位置会发生剧变。例如，朱元璋会从濠州转移到应天；郭子兴死后，其部众会归附朱元璋。
    *   **个人事件**：特定角色在满足条件后会“出山”或“移动”。比如你“三顾茅庐”式地拜访刘伯温，他会答应出山，并移动到你所在的城池。

---

### 💎 总结

这套城镇体系的设计，不只为了让地图更丰富。都城、省会、府城、县城的等级差异，对应着游戏里身份跃迁的阶梯；不同设施的分布，引导着玩家选择武将、文官、商人等人生路线；而人物的动态移动，则让每一次选择都可能改变命运。这三者相辅相成，让玩家在“修文习武、经世济民、一统天下”的沉浸式体验中，找到属于自己的路。
