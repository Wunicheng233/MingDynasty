# 08 - 合战系统

## 一、基础机制（借鉴太阁4合战）

### 1.1 触发场景

- 执行军务主命（出征/攻城）
- 历史剧情战役（鄱阳湖、北伐）
- 敌方入侵

### 1.2 参战单位

- **总大将**：主将，能力影响全军士气
- **备队**：最多4名副将，每将带一支部队
- **兵种**：每支部队可配置不同兵种

### 1.3 胜负条件

- **野战/水战**：敌方总大将兵力归零 **或** 士气归零 **或** 总大将退却
- **攻城战**：满足以上 **或** 城防御度归零
- 我方总大将退却 → 失败

### 1.4 回合流程（最多30回合，打到一方兵力/士气耗尽为止）

1. **手牌调整**：可弃3张，弃掉后保留剩余手牌
2. **秘策发动**：可将一张手牌换为已学会的任意战术卡（兵法等级决定次数）
3. **出牌**：**要么出1张战术卡，要么出1-3张基础卡，不能同时出**
4. **判定**：比较组合强弱，优先级高一方主动进攻
5. **结算**：计算伤害、士气、城防 → 下一回合

#### 抽牌规则
- 第一回合：初始抽5张
- 后续每回合：保留上回合剩余手牌，**再抽 `(对应兵种技能等级 + 1)` 张**

## 二、兵种体系（明朝特色）

| 兵种 | 特点 | 克制 | 被克 | 受技能 | 明朝背景 |
|------|------|------|------|------|---------|
| **步兵** | 防御稳定，成本低 | 弓兵 | 骑兵 | 步战 | 明军主力 |
| **骑兵** | 高机动高攻击 | 步兵 | 火器/弓兵 | 骑战 | 明初缴获元军马匹组建 |
| **弓兵** | 远程先手削弱 | 骑兵 | 步兵 | 弓术 | 明军重视弓弩 |
| **火器兵** | 远程高伤，雨天减半 | 骑兵/步兵 | - | 火器 | 明朝神机营，沐英首创三段击 |
| **水军** | 仅水战可用 | - | - | 水战 | 鄱阳湖之战奠定水师 |

**兵种相克**：克制方伤害×1.3，被克方×0.7。

**能力计算公式**：
```
部队攻击力 = 武将统率×0.3 + 武将武力×0.1 + 兵种基础攻击 × (1 + 技能等级×0.1)
部队防御力 = 武将统率×0.2 + 兵种基础防御 × (1 + 技能等级×0.05)
```

## 三、卡片与组合（太阁4核心）

### 3.1 基础卡组合强弱

```
三同张 (1.8倍) > 三连张 (1.5倍) > 二同张 (1.2倍) > 无组合 (1.0倍)
```

**数字大小影响**：
- 三同张：数字越小越强 → 111最强，999最弱
- 三连张：起始数字越小越强 → 123最强，789最弱

**相克循环**：
```
鱼鳞 → 雁行 → 锋矢 → 衡轭 → 方圆 → 长蛇 → 偃月 → 鹤翼 → 鱼鳞
```

## 五、伤害计算与攻防规则

### 5.1 优先级与攻防顺序

- **优先级规则**：战术卡(5) > 三同张(4) > 三连张(3) > 二同张(2) > 无组合(1)
- **优先级高 → 主动进攻**：使用本回合出牌正常计算，**全额伤害**
- **优先级低 → 防守反击**：**不使用本回合出牌计算伤害**，按公式：
  ```
  防守伤害 = 进攻伤害 × (防守方兵力 / 进攻方兵力) × (防守方士气 / 100) × 0.5
  ```
  防守伤害天然偏低，仅和兵力士气相关，符合真实战争场景

### 5.2 基础伤害公式

```
基础伤害 = (攻击方兵力 × 攻击方攻击力) / (防御方防御力 × 0.5 + 120)
实际伤害 = 基础伤害 × 组合系数 × 战术倍率 × 兵种相克 × 阵型相克
```

**士气影响**：
- 士气 ≥ 80：攻击+10%，暴击+5%
- 50~79：正常
- 30~49：攻击-10%，防御-10%
- 士气 < 30：攻击-20%，防御-20%，每回合概率溃散

## 六、水战特殊规则（鄱阳湖还原）

- 可用兵种：仅水军、弓兵、火器
- 火攻伤害×1.5
- 风向影响：上风方火攻伤害+20%，先手率+
- 铁索连环（陈友谅专属）：防御+30%，但火攻伤害×2.5 → 完全还原历史

## 七、历史战役还原

### 鄱阳湖之战（1363）

- 类型：水战
- 我方：朱元璋+徐达+常遇春+俞通海
- 敌方：陈友谅，兵力占优，铁索连环阵
- 特殊：陈友谅防御+30%，火攻伤害×2.5；我方剧情获得"鄱阳湖火攻"卡，伤害×3
- 胜利奖励：陈友谅人物卡，大量功勋，解锁水军高级修行

### 北伐大都（1368）

- 类型：野战+攻城
- 敌方：元顺帝、王保保
- 特殊：元军骑兵多，需用衡轭阵克制
- 胜利奖励：王保保人物卡，"北伐突进"秘传卡（徐达），触发明朝建立剧情
