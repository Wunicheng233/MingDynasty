## 《洪武立志传》社交与经济系统完整设计

社交与经济系统是《洪武立志传》中维系世界活力的“血脉”与“人际网络”。前者决定玩家与数百名历史人物的亲疏远近，直接影响拜师学艺、招揽人才、触发剧情、结盟联姻等核心玩法；后者则为玩家提供财富积累的多种路径，从贩夫走卒的低买高卖，到掌控盐引边贸的豪商巨贾，再到以全国赋税支撑帝国运转的财政大臣——经济路线本身就是一种完整的人生。

以下从社交系统（亲密度、互动方式、人际关系网、婚姻与结义）和经济系统（货币与物价、交易与投资、明朝特色商贸、身份与经济权限）两大板块展开，最后阐述两系统与全局的耦合及文字游戏实现建议。


### 一、社交系统

#### 1.1 亲密度体系

亲密度是衡量玩家与其他角色关系的核心数值，范围从 **-4（不共戴天）** 到 **+5（生死相随）**。每 ±1 对应一个“心级”图标，零为中立。

| 心级 | 数值范围 | 关系状态 | 可解锁内容 |
|------|---------|---------|-----------|
| -4 |  -100 ~ -80  | 不共戴天 | 见面可能触发战斗 |
| -3 |  -79 ~ -60  | 仇敌 | 拒绝一切互动 |
| -2 |  -59 ~ -30  | 交恶 | 不可修行、不可劝诱 |
| -1 |  -29 ~ -1   | 冷淡 | 可送礼，但效果差 |
| 0  |    0 ~ 19   | 中立 | 初次见面 |
| 1  |   20 ~ 39   | 一面之缘 | 可进行基础社交 |
| 2  |   40 ~ 59   | 略有交情 | **可邀请修行**、可劝诱 |
| 3  |   60 ~ 79   | 推心置腹 | **可获得人物卡**、可结义 |
| 4  |   80 ~ 94   | 莫逆之交 | 劝诱成功率极高 |
| 5  |   95 ~ 100  | 生死相随 | 永不叛变（配偶/结义兄弟） |

**亲密度变化规则**：
- **自然衰减**：每半年无互动，亲密度 -2（最低降至0）。
- **送礼**：根据礼物价值与对方喜好，增加 5~25 点。
- **茶会/宴饮**：成功则增加 10~15 点，失败可能扣减。
- **切磋**：胜利增加 5 点（对方性格豪勇则 +10），失败增加 2 点（对方指导）。
- **共同执行主命**：成功完成主命后，同队成员间亲密度各 +8。
- **在合战中救援**：若我方武将陷入危机，玩家救援成功则亲密度 +20。
- **负面事件**：如背后诋毁（被对方得知）、劝诱失败、背叛势力等大幅降低亲密度。

#### 1.2 互动方式

玩家在城镇中遇到其他角色时，可进行以下社交行动（每次消耗1天）：

**（1）送礼**
- 玩家从持有的宝物卡中选择一件赠予对方。
- 对方根据宝物价值、自身喜好（武将喜武器、文人喜书画、茶人喜茶器）以及当前亲密度，决定接受或婉拒。
- 接受则亲密度上升，宝物卡转移给对方；婉拒则无变化（但可再次尝试）。

**（2）茶会 / 宴饮**
- 邀请对方至茶室或酒楼，消耗少量金钱（茶会5贯，宴饮15贯）。
- 触发一个简短的“礼制”或“口才”检定（随机问答），成功则亲密度上升，失败则对方不悦（-2）。
- 宴饮效果更强，但需对方有“好酒”属性。

**（3）切磋（个人战）**
- 请求与对方进行武艺切磋。对方同意需满足：亲密度≥1心，且对方性格非“慎重”或“胆小”。
- 进入个人战系统。胜负均增加亲密度，胜利增加更多。
- 若玩家拥有对方人物卡且满足条件，切磋胜利可能习得其秘传卡。

**（4）交谈**
- 基础互动，可能触发以下子选项：
  - **打听情报**：了解某城镇物价、某武将下落。
  - **邀约同行**：邀请对方一同修行或执行主命（需亲密度≥2）。
  - **请求指导**：请求对方传授技能（需对方该项技能等级高于玩家且亲密度≥2）。
  - **劝诱**：说服对方加入自己势力（需亲密度≥2，且对方无主或对现主君不满）。

**（5）结义 / 婚姻**
- 特殊互动，需满足极高亲密度（4心以上）及特定条件（详见1.4）。

#### 1.3 人际关系网络与明朝特色

游戏中的角色关系不仅取决于玩家，NPC之间也存在复杂的亲疏网络，这会影响势力内部的政治斗争、劝诱成功率等。

**明朝特色关系网**：
- **同乡之谊（淮西集团）**：朱元璋的同乡（徐达、汤和、李善长等）相互间有天然亲密度加成（+20）。玩家若为凤阳出身，与淮西将领互动有优势。
- **师生之谊**：科举中榜后，考官与同年进士间形成师生、同年关系，官场相互提携。
- **姻亲网络**：通过婚姻可结成政治联盟。朱元璋娶马秀英获得郭子兴信任，是核心剧情。
- **厂卫监视**：若玩家得罪锦衣卫或东厂，与某些官员的亲密度提升会受到暗中阻碍（送礼效果减半）。

#### 1.4 婚姻与结义系统

**婚姻**：
- **条件**：
  - 双方异性，亲密度达到4心以上。
  - 玩家身份至少为“千户”或城主。
  - 部分剧情角色（如马秀英）有特定事件触发。
- **流程**：
  - 向对方求婚（需准备聘礼，价值至少100贯的宝物）。
  - 对方同意后，举办婚礼（消耗时间3天，金钱200贯）。
  - 婚后配偶将移居至玩家所在城镇，可触发居家对话、获取内政建议、偶尔赠送宝物。
- **效果**：
  - 亲密度锁定为5心，永不衰减。
  - 配偶所属势力若与玩家敌对，可能触发特殊剧情（如马秀英在郭子兴死后追随朱元璋）。

**结义兄弟**：
- **条件**：
  - 双方同性，亲密度达到4心以上。
  - 双方身份差距不大（非主君与家臣）。
  - 最多可拥有2名结义兄弟。
- **流程**：
  - 提出结义，对方同意后举行结拜仪式（消耗1天，金钱50贯）。
  - 效果：亲密度锁定为5心，永不背叛；合战时若在同一军团，士气额外+10。

#### 1.5 社交系统与卡片、主命的耦合

- 亲密度达到3心 → **获得该角色的人物卡**。
- 亲密度达到2心 → 可邀请对方**共同修行**（两人一同进行小游戏，双方技能经验均增加）。
- 亲密度影响**主命“劝诱”的成功率**，以及作为君主时**属下忠诚度**。
- 亲密度影响**合战“招降”战术卡**的效果。


### 二、经济系统

#### 2.1 货币与物价体系

明朝初期货币体系复杂，游戏简化为三种基本货币单位：

| 货币 | 换算关系 | 主要用途 |
|------|---------|---------|
| **文** | 基本单位 | 小额交易（米、盐） |
| **贯** | 1贯 = 1000文 | 日常买卖、军资金、俸禄 |
| **两（银）** | 1两 ≈ 1.5贯（浮动） | 贵重宝物、大宗贸易、盐引 |

**物价波动机制**：
- 每座城镇有独立的物价指数，受以下因素影响：
  - **丰歉**：丰收年粮价下跌，灾年上涨。
  - **战乱**：交战状态下所有物价上涨20%。
  - **季节**：秋收后粮价最低，春荒时最高。
  - **商路阻断**：若某城镇被围城或海禁加强，特产价格飙升。
- 玩家“商政”技能等级越高，在交易界面看到的“建议买入/卖出”提示越准确。

#### 2.2 基本交易与投资

**（1）市集交易**
- 每个城镇的“市集”中可买卖当地特产及基础物资（粮、盐、铁、马等）。
- 玩家可查看各城镇间物价差，通过低买高卖赚取利润——这是商人路线初期的核心玩法。
- 界面示例：
```
═══════════════════════════════════════════
            应天府 · 市集
═══════════════════════════════════════════
持有金钱：320贯

商品        库存    买入价      卖出价
大米        2000石   1.2贯/石   0.8贯/石
丝绸        500匹    8贯/匹     6贯/匹
茶叶        800斤    3贯/斤     2贯/斤
盐          300引    15贯/引    12贯/引（需盐引凭证）

1. 买入  2. 卖出  3. 查看其他城镇物价（需密探情报）  4. 离开
═══════════════════════════════════════════
```

**（2）城镇投资**
- 向市集“投资”可提升该城镇的“投资等级”，解锁更高级特产、降低物价波动。
- 投资消耗金钱，且需“商政”技能达到一定等级。
- 投资效果：每投资100贯，投资经验+1，城镇繁荣度+1%。繁荣度影响该城税收和人口。

#### 2.3 明朝特色商贸机制

**（1）开中法·盐引贸易**
- **背景**：明初为解决边关军粮运输，允许商人向边关运送粮食，换取“盐引”凭证，再凭盐引到两淮、长芦等盐产区支取食盐贩卖。
- **游戏实现**：
  - 玩家在边关城镇（如大同、宣府）的“军仓”可接取“运粮换引”任务。
  - 运送一定数量粮食（消耗时间与运费）后，获得对应数量的“盐引”宝物卡。
  - 持盐引前往产盐区（扬州、淮安）的“盐运司”，可将盐引兑换为食盐，再运往内地销售。
  - 食盐是游戏中最暴利的商品之一，但无盐引私自贩盐属于“私盐”，被查获将面临罚款或牢狱之灾。
- **数据关联**：商政技能越高，每张盐引能换取的食盐量越多；口才技能影响与盐运司官员的交涉。

**（2）海禁与走私**
- 明朝实行严厉海禁，仅允许官方朝贡贸易。但闽浙沿海走私猖獗。
- 玩家可选择**合法海商**路线（需获得“市舶司”许可，身份至少为指挥使）或**走私**路线（风险高回报极高）。
- 走私时需进行“密探”潜入检定，成功则获利数倍，失败则损失货物并被通缉（降低与该势力的外交关系）。

**（3）宝钞与通货膨胀**
- 明初发行“大明宝钞”纸币，但不久便严重贬值。
- 游戏中，随着年份推移（进入洪武中后期），宝钞的实际购买力逐年下降。玩家若持有大量宝钞，需及时兑换为白银或实物。
- 这一设定增加了经济运营的策略深度。

**（4）商帮会馆**
- 在扬州、苏州、太原等商业重镇，有晋商、徽商、闽商等商帮的会馆。
- 玩家可申请加入商帮（需商政Lv2，交纳会费）。加入后享受：
  - 会馆内可获取各地精准物价情报。
  - 可承接商帮专属主命（如“运送漕粮”、“承办边饷”）。
  - 可向会馆借贷应急资金。
- 商帮内也有亲密度系统，与商帮首领关系好可晋升为“会首”，掌控更大商业网络。

#### 2.4 身份与经济权限

玩家的身份直接影响可从事的经济活动：

| 身份阶段 | 可从事经济活动 |
|---------|---------------|
| 平民/浪人 | 市集倒卖、打工（小游戏）、借贷 |
| 家臣（百户以下） | 领取俸禄（微薄）、执行“筹措军资金”主命 |
| 家臣（千户以上） | 可受封领地，获得领地税收；参与开中法 |
| 城主/国主 | 管理城池金库、制定税率、投资城镇、发行军令调配物资 |
| 君主 | 掌控全国财政，决定赋税政策（轻徭薄赋或重税敛财）、发行宝钞、与外国朝贡贸易 |

**俸禄与税收**：
- 家臣俸禄：每两月评定会发放，数额由身份等级决定（百户约20贯，五军都督约300贯）。
- 城主税收：每季度从所辖城镇征收一次，总额 = （城镇繁荣度 × 税率） - 维持费。玩家可调整税率，但过高会降低民忠，引发暴动。
- 君主国家财政：全国税收总和 + 朝贡收入 - 官员俸禄 - 军费。若长期赤字，军队士气下降，可能发生叛乱。

#### 2.5 经济系统与主命、卡片的耦合

- **主命“筹措军资金”**：调用商政小游戏，成功后增加城池军资金。
- **主命“买卖兵粮”**：调用交易界面，需低买高卖完成指标。
- **宝物卡**：可通过购买、任务奖励、战胜敌将获得，用于送礼或装备提升属性。
- **称号“豪商”**：个人资产达到10万贯获得，魅力+5，商人路线终极称号之一。

**关键耦合点说明**：
- **社交 → 卡片**：亲密度3心获得人物卡。
- **社交 → 主命**：劝诱、外交类主命成功率受亲密度影响。
- **经济 → 社交**：送礼需要金钱购买宝物；茶会/宴饮消耗金钱。
- **经济 → 主命**：军资主命完成效果受商政技能影响。
- **经济 → 时间**：物价每月1日刷新；投资效果随时间逐渐显现。
- **社交 → 合战**：结义兄弟在同一军团有士气加成。


### 四、文字游戏实现建议

**1. 社交界面设计**
```
═══════════════════════════════════════════
            与 徐达 交谈
═══════════════════════════════════════════
徐达（元帅） 亲密度：♥♥♥♡♡（推心置腹）

1. 送礼
2. 邀请茶会
3. 切磋武艺
4. 打听情报
5. 请求指导兵法
6. 劝诱（需亲密度≥2心）
7. 结义（需亲密度4心）
8. 离开

请输入数字选择：
═══════════════════════════════════════════
```

**2. 经济数据存储（JSON）**
```json
{
  "player_assets": {
    "gold": 12500,           // 文，游戏内统一用文存储，显示时换算
    "silver_tael": 800,      // 两
    "treasure_cards": ["TREASURE_SWORD", "TREASURE_TEA"],
    "salt_tickets": 5        // 持有的盐引数量
  },
  "city_economy": {
    "YINGTIAN": {
      "prosperity": 78,
      "tax_rate": 0.15,
      "prices": { "rice": 1200, "silk": 8000, "salt": 15000 },
      "investment_level": 3
    }
  }
}
```

**3. 物价计算函数（示例）**
```python
def get_city_price(city_id, goods_id):
    base_price = GOODS_BASE[goods_id]
    city_data = world_state["cities"][city_id]
    # 繁荣度影响：越繁荣，物价越平稳（偏离系数小）
    # 季节、战乱、投资等级等因素叠加
    return round(base_price * city_data["price_modifier"][goods_id])
```

**4. 亲密度变化事件总线**
建议使用事件订阅机制，当合战胜利、主命完成、节日到来时，向社交系统发送事件，统一处理亲密度变化。例如：
```python
EventBus.emit("battle_won", winner=player, loser=chen_youliang)

```

通过以上设计，《洪武立志传》的社交与经济系统既保留了《太阁立志传》系列“送礼喝茶攒人脉、低买高卖赚身家”的经典乐趣，又深度融入了明朝特有的社会结构与财政制度。玩家的人际网络将直接影响仕途与争霸，而经济实力则是一切行动（招兵、送礼、投资）的基础——两系统共同支撑起一个真实可感的明初世界。
