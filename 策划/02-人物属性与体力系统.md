## 三、体力系统（借鉴太阁4经典设计）

### 3.1 基础定义

- **当前体力**：0 ~ 最大体力，归零则昏迷
- **最大体力** = 基础体力 + 称号加成 + 宝物加成
- **基础体力** = 60 + (武力 × 0.8)

示例：
- 朱元璋武力78 → 基础 = 60 + 62.4 = **122**
- 常遇春武力100 → 基础 = 60 + 80 = **140**

### 3.2 体力状态层级

| 体力百分比 | 状态 | 效果 |
|-----------|------|------|
| 100% ~ 75% | 精力充沛 | 修行效率 +10% |
| 75% ~ 40% | 略有倦意 | 移动速度 -10% |
| 40% ~ 20% | 疲惫不堪 | 移动-20%，经验-30%，个人战手牌-1 |
| 20% ~ 5% | 体力透支 | 无法修行/出征，个人战武力-20% |
| < 5% | 危在旦夕 | 无法执行多数指令，10%/天概率昏迷 |

**生病状态**：独立异常状态，体力上限-30%，自然恢复停止，必须医治。

### 3.3 体力消耗表

| 行动 | 消耗 | 说明 |
|------|------|------|
| 城镇内设施移动 | 0 | 不消耗 |
| 修行（5天） | 8 ~ 12 | Lv越高消耗越多 |
| 执行主命（全程） | 10 ~ 30 | 难度决定 |
| 跨省移动 | 5 | 骑战每级减1，最低2 |
| 宴饮/茶会 | 3 |  |
| 武馆切磋 | 战后损失×50% |  |
| 自然消耗 | 1/10天 | 精力充沛时减半 |

### 3.4 体力恢复表

| 方式 | 恢复量 | 消耗 | 限制 |
|------|-------|------|------|
| 旅店住宿 | 恢复满 | 50文 × 恢复百分比 | 消耗5天，生病不能全恢复 |
| 自宅休养 | 恢复满 | 无 | 消耗5天，需拥有自宅 |
| 医馆治疗 | 恢复满+解除生病 | 200文 | 消耗3天 |
| 金疮药 | +30 | 消耗物品 | 任何时候 |
| 续命丹 | +50 | 消耗物品 | 任何时候 |
| 与妻子互动 | +15 | 无 | 每天1次 |
| 自然恢复 | +2/天 | 无 | 非生病状态 |
| 旅店/自宅休养 | +5/天 |  | 休养期间 |

### 3.5 伤病与死亡机制

- **体力归零昏迷**：
  - 城镇内：被救起，消耗7天，恢复到20%
  - 野外孤身：30%概率直接死亡，70%被救但损失50%金钱
- **久病不愈**：生病超过30天 → 最大体力永久-5
- **角色死亡**：
  - 有成年继承人/结义兄弟 → 可继承继续游戏
  - 无继承人 → 游戏结束

### 3.6 明朝特色设计

1. **勤政过劳**（君主专属）：
   - 连续处理政务超过20天 → 累积过劳值
   - 满了触发"勤政猝晕" → 强制休息7天，最大体力永久-5
   - 致敬朱元璋工作狂历史形象

2. **丹药与丹毒**：
   - 丹药瞬间回满体力，但增加丹毒
   - 丹毒过高 → 最大体力永久下降，甚至暴毙
   - 契合明朝皇帝迷信丹药的历史

3. **缠足与女性体力**：
   - 女性角色初始体力上限-10%，移动消耗+20%
   - 可通过事件"放足"解除，呼应历史进步

## 四、人际关系网络（明朝特色）

### 4.1 初始关系加成

- **淮西集团**：朱元璋同乡（徐达、汤和等）相互间+20亲密度
- **师生同年**：科举中形成，官场相互提携，亲密度提升更快
- **姻亲政治**：婚姻结成政治联盟，亲密度锁定
- **敌对仇恨**：朱元璋-陈友谅初始-3亲密度

### 4.2 与其他系统耦合

- 亲密度2心 → 可拜师修行
- 亲密度3心 → 获得人物卡
- 亲密度影响劝降成功率
- 结义兄弟合战同队 → 士气+10
