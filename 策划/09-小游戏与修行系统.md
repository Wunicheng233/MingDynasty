# 09 - 小游戏与修行系统

## 一、设计理念（继承太阁4）

太阁4的核心乐趣之一：**每项技能对应一个小游戏**，修行就是玩小游戏，胜利获得经验，升级解锁技能。我们将图形界面小游戏**纯文字化改造**，保留策略深度，加入明朝文化特色。

### 核心循环

```
玩家找到师父（亲密度≥2心）
    ↓
选择师事修行 → 进入对应技能的小游戏
    ↓
小游戏结果判定 → 完胜/小胜/失败
    ↓
完胜 → 技能经验+2，**解锁对应卡片**
小胜 → 技能经验+1
失败 → 无经验
    ↓
经验累积满 → 技能等级+1 → 解锁更多阵型/战术/手牌
```

每次修行消耗 **5天** 时间，**8~12点**体力（技能等级越高消耗越多）。

## 二、16项技能对应16款小游戏（完整本土化）

### 2.1 步战 → 象棋残局杀

**核心机制**：中国象棋一步杀残局，红先黑死，找出正确杀棋要点。

交互规则：先点击你要移动的红色棋子选中，再点击目标空位移动，一步杀即过关。

- Lv1：只出基础死活，经典杀法（卧槽马、重炮、铁门栓）
- Lv2：开放中等难度（马后炮、白脸将）
- Lv3：开放全难度（钓鱼马、双马饮泉）

每次随机出 **3 题**，答对越多奖励越高。

**典故**：象棋古称"象戏"，是中国传统军事谋略训练工具，"局"就是棋盘，对应战场局势，一步杀对应步兵突击精准一击。朱元璋起兵濠梁，象棋锻炼谋略。

### 2.2 骑战 → 赛马争先

**核心机制**：骰子竞速，到终点累计15点。

每回合选择：
- 稳扎稳打：正常掷骰，无惩罚
- 策马加鞭：掷骰+1，但出1则下回合不能行动
- 以身挡道：不前进，对手掷骰-2

骑战等级越高，"策马加鞭"受惊概率越低。**先到终点** → 完胜。

**典故**：常遇春单骑突阵，模拟骑兵训练。

### 2.3 火器 → 神机校射

**核心机制**：风向+距离影响瞄准，5次射击累计得分。

- 距离近/中/远 → 不同命中要求
- 顺风+1，逆风-1，侧风不变
- 头部3分，躯干2分，腿部1分

积分≥8 → 完胜，≥5 → 小胜。火器等级越高，骰子面数从d6升到d8，命中率提升。

**典故**：神机营是明朝最早的专业火器部队，沐英首创三段击。

### 2.4 水战 → 楼船破浪

**核心机制**：资源分配，15段航程，初始船工体力10，水军士气10。

每回合选择：
- 全速前进：+2段，消耗士气2，可能遇暗礁
- 中速巡航：+1段，消耗体力1
- 停船休整：不前进，恢复3点体力或士气

任意资源归零 → 失败；两种资源≥5到达终点 → 完胜。

**典故**：鄱阳湖之战，朱元璋水军突破陈友谅。

### 2.5 弓术 → 百步穿杨

**核心机制**：6支箭，每支选择拉力，随机目标距离。

- 轻拉：近必中，中50%，远0%
- 中拉：近必中，中70%，远30%
- 满拉：近50%，中70%，远60%

6箭全中 → 完胜，4-5中 → 小胜。弓术等级越高，满拉命中率越高。

**典故**：明朝武举射箭必考，名将花云善射。

### 2.6 武艺 → 武馆切磋

**核心机制**：直接调用个人战系统，与师父/同门切磋。

**优势**：复用现有系统，无需额外开发。胜负都给经验，完胜给双倍。

**典故**：朱元璋皇觉寺习武，太祖长拳流传后世。

### 2.7 兵法 → 围棋死活

**核心机制**：五道围棋死活题，找出杀棋/活棋要点。

- Lv1：简单题，3题对 → 完胜
- Lv2：中等题，4题对 → 完胜
- Lv3：难题，5题对 → 完胜

**典故**：朱元璋、刘伯温都爱好围棋，刘基棋艺高超。围棋锻炼谋略，契合兵法。

### 2.8 农政 → 劝课农桑

**核心机制**：资源分配开垦荒地，10块地，初始资金50贯。

每回合：
- 开垦荒地：花钱开垦一块
- 修建水利：花钱提升所有地产出
- 招募佃农：花钱增加人力

随机事件：旱灾/蝗灾/丰收影响产出。

全开垦完剩余资金≥20贯 → 完胜。农政等级越高，初始资金越多，受灾概率越低。

**典故**：朱元璋出身农家，立国后推行"广积粮"。

### 2.9 工政 → 筑城考工

**核心机制**：用不同重量石块，让总重量尽可能接近目标重量。

石块：大块500，中块250，小块50，Lv2解锁20小块，Lv3解锁凿子修整（可敲掉90-100）。

误差≤20 → 完胜，≤50 → 小胜。

**典故**：明初修建南京都城城墙，世界最长，砖石刻工匠名字追责。

### 2.10 商政 → 珠算商贾

**核心机制**：6道算术题，快速心算回答。

题目：单价×数量、折扣计算、钱币换算、利润计算。

全对 → 完胜，对4-5 → 小胜。商政等级越高，限时越宽松但题目越难。

**典故**：明朝商品经济发达，晋商徽商崛起，开中法盐引贸易需要快速计算。

### 2.11 律政 → 明律断案

**核心机制**：原告被告证人证词，找出矛盾，选择正确《大明律》条目。

找出矛盾+选对律条 → 完胜 → 对一个 → 小胜。等级越高，矛盾越隐蔽，证词越长。

**典故**：朱元璋亲自主持修订《大明律》，是中国法制史上重要法典。

### 2.12 口才 → 游说纵横

**核心机制**：根据对方情绪状态选择话术，6回合内将情绪从0升到100。

- 晓之以理：情绪平稳效果好，激动反而减
- 动之以情：情绪低落效果好，得意反而减
- 诱之以利：情绪摇摆效果好，警觉反而减

根据文字反馈判断情绪，选择对应话术。口才等级越高，初始情绪越高，负面效果越弱。

**典故**：朱元璋麾下谋士善游说，劝降纳叛。

### 2.13 礼制 → 朝仪习礼

**核心机制**：记忆礼仪顺序，演示后复原。

演示：迎神→奠玉帛→进俎→初献→...
玩家需要从打乱选项中选出正确顺序。

全对 → 完胜，对3-4 → 小胜。礼制等级越高，需要记忆的步骤越多。

**典故**：朱元璋命儒臣编订《大明集礼》，朝仪礼制森严。

### 2.14 医术 → 本草配方

**核心机制**：记忆药材抓取顺序，然后复现。

医师按顺序抓4-6味药，玩家记住顺序然后输入。

完全正确 → 完胜，对n-1 → 小胜。医术等级越高，需要记忆的药材越多。

**典故**：明代李时珍著《本草纲目》，集古代医药之大成。

### 2.15 密探 → 潜龙谍影

**核心机制**：路径选择+风险掷骰，5-7个节点到达终点。

每个节点三选一：
- 翻墙：掷≥4成功，失败警报+20%
- 正门：掷≥5成功，失败口才检定
- 狗洞：必定成功，但最后奖励降低
- 贿赂：花钱，必定成功

警报到100% → 失败。全程警报0到达 → 完胜。密探等级越高，成功需要点数越低。

**典故**：朱元璋建国后设立锦衣卫，监视百官，侦缉密探。

### 2.16 航海 → 天文定向

（补充，原有策划没写，补充完整16项）

**核心机制**：根据星辰方位判断航向，多次修正到达目标。

- 北极星方位判断纬度，选择航向
- 每次选择后告诉你偏北/偏南/正确
- 5次修正内到达 → 完胜

**典故**：郑和下西洋，天文航海技术成熟。

## 四、方便玩家设计：跳过机制

参考太阁4，技能达到**Lv3**后，再次修行可以选择**跳过小游戏**：
- 跳过直接获得小胜奖励（经验+1）
- 但无法解锁卡片（卡片需要首次完胜解锁）
- 照顾不喜欢玩小游戏的玩家，同时保留"完胜解锁"的收集激励

## 五、与太阁立志传4对比

| 太阁4 | 洪武立志传 |
|------|------------|
| 16项技能对应16小游戏 | 16项技能对应16小游戏（全部文字化+本土化） |
| 图形交互 | 文字交互，适合文字游戏/网页 |
| 运茶/采矿/算术... | 全部改造为符合明朝背景的新设计 |
| 完胜解锁新卡片 | 完胜解锁新卡片（完全继承） |
| 可跳过小游戏 | 可跳过小游戏（完全继承） |

我们保留了太阁4"一项技能一个小游戏"的核心乐趣，把所有图形小游戏改造为纯文字可玩，同时全部注入明朝历史文化元素。
