## 📖 项目简介

《洪武立志传》是一款**文字网页策略游戏**，灵感来源于经典游戏《太阁立志传4》，将背景本土化到**元末明初**，玩家可以扮演朱元璋以及后来的明朝开国功臣，从濠州起兵开始，一步步完成统一中国的大业。

游戏继承了太阁系列的核心玩法：
- 🏃 **高自由度** - 修行、任务、社交、收集，你选择你的道路
- 🎴 **卡片驱动** - 一切可收集内容均卡片化，600+卡片等你收集
- ⚔️ **个人战系统** - 数字组合+特殊武技+名将秘传，策略深度拉满
- 🌾 **小游戏修行** - 16项技能对应16个不同小游戏，修行就是玩儿！

## ✨ 游戏特色

### 已实现功能

- ✅ **16个小游戏** - 每项技能对应一个独特小游戏，纯文字点击交互
  - 步战「步军演武」、骑战「赛马争先」、火器「神机校射」、水战「楼船破浪」
  - 兵法「排兵布阵」、武艺「武馆切磋」、农政「劝课农桑」、工政「筑城考工」
  - 商政「珠算商贾」、律政「明律断案」、口才「游说纵横」、礼制「朝仪习礼」
  - 医术「本草配方」、密探「潜龙谍影」、航海「罗盘导航」、文墨「文墨填空」

- ✅ **完整个人战系统** (太阁4核心机制本土化)
  - 数字卡 1-9 + 特殊技卡 + 名将专属秘传卡
  - 12个**明朝特色数字组合技** (洪武定鼎/常山蛇势/八卦游身等)
  - 优先度判定系统，高优先级者先行动，低优先级本回合无法行动
  - **全效果完整实现**：回血、闪避反击、完全防御、无视防御、不反击、后发先至、倍率伤害
  - 抽牌规则：保留未出牌，每回合抽「武艺等级」张，手牌上限9张

- ✅ **模块化核心架构** (2025重构完成)
  - 单一职责拆分，消除千行大文件，便于多人协作开发
  - `base/` - 小游戏基类 `BaseGame`，通用生命周期逻辑复用
  - `managers/` - 场景管理器/动画管理器/结果结算管理器/小游戏初始化器
  - `systems/` - 卡片系统/社交系统/技能系统/存档系统，职责分离
  - `modules/` - 各个场景视图渲染器拆分，每个文件只负责一个场景
  - `utils/` - 通用工具函数，消除重复代码

- ✅ **完整卡片收集系统** - 图片化UI美化完成
  - 人物卡使用名人立绘，技能卡使用专属插画，称号卡使用专属题图
  - 一切可收集内容均卡片化，目前已定义 **200+张卡片**，规划600+张
  - 卡片收集进度跨存档继承

- ✅ **社交系统** - 完整亲密度关系网
  - 11+核心元末明初历史人物，可互动送礼/茶会/切磋/结义/结婚
  - 亲密度范围 -4 ~ +5，对应不同互动解锁
  - 亲密度达到3心自动解锁人物卡，可在新游戏选择扮演

- ✅ **合战系统** - 核心玩法循环完整
  - 支持野战/攻城战/水战三种模式
  - 完整兵种相克、阵型选择、秘策系统、AI决策
  - 胜利统计解锁称号

## 🚀 如何运行

### 方法一：直接打开（最简单）

1. 克隆项目
```bash
git clone https://github.com/Wunicheng233/MingDynasty.git
cd MingDynasty
```

2. 双击根目录的 `start-game.bat`，即可自动在默认浏览器中打开 `index.html`。

3. 你也可以直接用浏览器打开 `index.html`。

4. 开发调试可以打开 `debug-minigames.html` 直接测试所有小游戏

### 方法二：本地服务器

如果你有Python，可以启动本地服务器：
```bash
# Python 3
python -m http-server 8000
# 然后访问 http://localhost:8000
```

## 🎮 玩法简介

### 开局

- 第一章默认扮演**朱元璋**，从濠州起兵开始
- 初始身份：红巾军士兵，通过完成任务获得功勋晋升
- 每次修行/任务消耗天数，时间推进，每两月一次评定会

### 成长

- 在城镇武馆/作坊找到师父，**师事修行**对应技能
- 通过小游戏考核，获得技能经验，经验满了升级
- 首次**完胜**小游戏可以解锁对应技能卡片
- 卡片收集**跨存档继承**，解锁的人物可以在新游戏选择扮演

### 战斗

- **个人战**：武馆切磋、江湖遭遇、武举考核都会触发
- 每回合抽牌，可选 1张特殊技 + 最多3张数字卡
- 数字卡可以组合出明朝特色组合技
- 优先度高的先攻击，一方体力归零则战斗结束
- 胜利可以获得技能经验，有概率习得对手的秘传卡
