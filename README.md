## 📖 项目简介

《洪武立志传》是一款**文字网页策略游戏**，灵感来源于经典游戏《太阁立志传4》，将背景本土化到**元末明初**，玩家可以扮演朱元璋以及后来的明朝开国功臣，从濠州起兵开始，一步步完成统一中国的大业。

游戏继承了太阁系列的核心玩法：
- 🏃 **高自由度** - 修行、任务、社交、收集，你选择你的道路
- 🎴 **卡片驱动** - 一切可收集内容均卡片化，600+卡片等你收集
- ⚔️ **个人战系统** - 数字组合+特殊武技+名将秘传，策略深度拉满
- 🌾 **小游戏修行** - 16项技能对应16个不同小游戏，修行就是玩儿！

## ✨ 游戏特色

### 已实现功能

- ✅ **16个小游戏** - 每项技能对应一个独特小游戏，纯文字点击交互
  - 步战「步军演武」、骑战「赛马争先」、火器「神机校射」、水战「楼船破浪」
  - 兵法「排兵布阵」、武艺「武馆切磋」、农政「劝课农桑」、工政「筑城考工」
  - 商政「珠算商贾」、律政「明律断案」、口才「游说纵横」、礼制「朝仪习礼」
  - 医术「本草配方」、密探「潜龙谍影」、航海「罗盘导航」、文墨「文墨填空」

- ✅ **完整个人战系统** (太阁4核心机制本土化)
  - 数字卡 1-9 + 特殊技卡 + 名将专属秘传卡
  - 12个**明朝特色数字组合技** (洪武定鼎/常山蛇势/八卦游身等)
  - 优先度判定系统，决定攻击顺序
  - 丰富的特殊效果：回血、闪避、完全防御、无视防御、反击、变速

- ✅ **模块化核心数据层**
  - 静态数据与动态状态分离，便于维护和扩展
  - 10+核心元末明初历史人物
  - 8个核心城镇（濠州/应天/滁州/江州/苏州等）
  - 5大势力（郭子兴军/朱元璋军/陈友谅/张士诚/元廷）
  - 150+张已定义卡片

## 🛠 技术架构

```
三层架构（按照策划文档设计）
┌─────────────────────────────────┐
│        玩法表现层 (UI)          │
│   HTML + CSS + JavaScript      │
└────────────┬────────────────────┘
             │
┌────────────▼────────────────────┐
│        系统模块层                │
│  GameState(状态管理) + GameView(渲染) │
└────────────┬────────────────────┘
             │
┌────────────▼────────────────────┐
│        核心数据层                │
│   静态模板库 + 动态游戏状态      │
```

**卡片驱动设计**：一切可收集内容（人物、技能、战术、称号、名所）均为卡片，统一由`card_id`标识，跨存档继承收集进度。

## 🚀 如何运行

### 方法一：直接打开（最简单）

1. 克隆项目
```bash
git clone https://github.com/Wunicheng233/MingDynasty.git
cd MingDynasty
```

2. 用浏览器直接打开 `index.html` 即可开始游戏

3. 开发调试可以打开 `debug-minigames.html` 直接测试所有小游戏

### 方法二：本地服务器

如果你有Python，可以启动本地服务器：
```bash
# Python 3
python -m http-server 8000
# 然后访问 http://localhost:8000
```

## 🎮 玩法简介

### 开局

- 第一章默认扮演**朱元璋**，从濠州起兵开始
- 初始身份：红巾军士兵，通过完成任务获得功勋晋升
- 每次修行/任务消耗天数，时间推进，每两月一次评定会

### 成长

- 在城镇武馆/作坊找到师父，**师事修行**对应技能
- 通过小游戏考核，获得技能经验，经验满了升级
- 首次**完胜**小游戏可以解锁对应技能卡片
- 卡片收集**跨存档继承**，解锁的人物可以在新游戏选择扮演

### 战斗

- **个人战**：武馆切磋、江湖遭遇、武举考核都会触发
- 每回合抽牌，可选 1张特殊技 + 最多3张数字卡
- 数字卡可以组合出明朝特色组合技
- 优先度高的先攻击，一方体力归零则战斗结束
- 胜利可以获得技能经验，有概率习得对手的秘传卡

游戏背景设定在**元至正十二年（1352年）**，朱元璋刚刚投奔郭子兴红巾军，龙椅从濠州坐到应天，最终一统天下。

游戏中所有人物、城镇、势力都基于真实历史改编，带你重走明朝开国之路。
